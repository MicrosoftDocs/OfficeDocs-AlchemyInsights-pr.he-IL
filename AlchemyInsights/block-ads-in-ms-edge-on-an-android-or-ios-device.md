# <a name="block-ads-in-microsoft-edge-on-an-adroid-or-ios-device"></a>חסימת מודעות ב-Microsoft Edge במכשיר Adroid או iOS

כדי לחסום את כל המודעות למעט אלה הנחשבות למקובלות מאת AdBlock Plus:
1. בחר **.** .. > **הגדרות**  >  **חוסמי תוכן**.
2. הפעלת מודעות לחסימות.

כדי לחסום את כל המודעות:
1. בחר **.** .. > **הגדרות**  >  **חוסמי תוכן**.
2. הפעלת מודעות לחסימות.
3. בחר **הגדרות מתקדמות**.
4. בטל את האפשרות אפשר מודעות מקובלות.
