# <a name="the-open-with-explorer-button-is-missing"></a>הלחצן ' פתח עם סייר ' חסר

הלחצן ' **פתח עם סייר** ' אינו מופיע בחוויה המודרנית של SharePoint ו-onedrive. כדי לפתוח ספריית מסמכים בסייר הקבצים, דפדף אליו ב-Internet explorer ובחר \> **בתצוגת תפריט תצוגה בסייר הקבצים**. האפשרות **תצוגה בסייר הקבצים** אינה זמינה ב-Microsoft Edge, Google Chrome או Firefox. 
  
> [!NOTE]
> הפתיחה באמצעות Explorer נתמכת רק באמצעות Internet Explorer. במקום זאת, [סנכרן קבצי SharePoint עם הלקוח החדש של סינכרון OneDrive](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx) המספק [קבצים לפי דרישה](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx). גש אל כל הקבצים שלך ב-OneDrive ללא שימוש בשטח אחסון מקומי.
