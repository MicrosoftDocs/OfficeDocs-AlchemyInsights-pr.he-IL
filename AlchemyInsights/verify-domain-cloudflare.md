# <a name="verify-your-domain-with-cloudflare"></a>אימות התחום שלך באמצעות Cloudflare

כדי לאמת את התחום שלך, יהיה עליך להוסיף רשומת TXT לרשם התחומים שלך כדי לוודא שהתחום שלך נמצא בבעלותך. 

תחילה, בצע את [אשף הוספת תחום](https://admin.microsoft.com/Adminportal#/Domains) על-ידי **בחירת** \> **הגדרות תחומים** ולאחר מכן **הוסף תחום**.
  
ראה [יצירת רשומות DNS ב- Cloudflare עבור Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/dns/create-dns-records-at-cloudflare) לקבלת הוראות מפורטות.
