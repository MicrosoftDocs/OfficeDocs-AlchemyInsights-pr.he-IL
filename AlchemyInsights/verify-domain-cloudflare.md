# <a name="verify-your-domain-with-cloudflare"></a>אימות התחום שלך באמצעות Cloudflare

כדי לאמת את התחום שלך, יהיה עליך להוסיף רשומת TXT לרשם התחומים כדי לאמת שהתחום נמצא בבעלותך. 

תחילה, בצע את [אשף הוספת התחום](https://admin.microsoft.com/Adminportal#/Domains) על-ידי בחירת תחומים של **הגדרות** \> **Domains**ולאחר מכן **הוסף תחום**.
  
ראה [יצירת רשומות DNS ב-Cloudflare עבור Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/dns/create-dns-records-at-cloudflare) לקבלת הוראות שלב-אחר-שלב.
