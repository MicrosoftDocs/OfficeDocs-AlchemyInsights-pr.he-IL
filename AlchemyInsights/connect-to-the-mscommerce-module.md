# <a name="connect-to-the-mscommerce-module"></a>התחברות למודול MSCommerce

עליך להיות מחובר למודול MSCommerce לפני שתוכל להציג או להגדיר את מדיניות AllowSelfServicePurchase.  

כדי להתחבר למודול MSCommerce, בשורת ההנחיה של PowerShell (PS C: \) , הזן את הפקודה הבאה:

`Connect-MSCommerce`

פעולה זו תפתח תיבת דו-שיח של כניסה. הזן את שם המשתמש והסיסמה שלך כדי להיכנס.

**הערה:** &nbsp; &nbsp; החשבון המשמש לכניסה חייב להיות מנהל חברה או חיוב.
