# <a name="edit-a-read-only-file"></a>עריכת קובץ לקריאה בלבד

בספריות מסמכים, בעת לחיצה על שם מסמך, תוכנית Office עשויה לפתוח אותה באופן בלתי צפוי כקובץ לקריאה בלבד. כדי לפתוח אותו לעריכה, לחץ על **פתח** (שלוש נקודות) ולאחר מכן לחץ על **ערוך.**
