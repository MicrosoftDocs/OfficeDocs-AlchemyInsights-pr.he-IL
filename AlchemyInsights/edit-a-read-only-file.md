# <a name="edit-a-read-only-file"></a>עריכת קובץ לקריאה בלבד

בספריות מסמכים, בעת לחיצה על שם מסמך, ייתכן שתוכנית Office תפתח אותה באופן בלתי צפוי כקובץ לקריאה בלבד. כדי לפתוח אותו לעריכה, לחץ על **פתח** (שלוש הנקודות) ולאחר מכן לחץ על **עריכה.**
