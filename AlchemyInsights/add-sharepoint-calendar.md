# <a name="add-a-sharepoint-calendar"></a>הוספת לוח שנה של SharePoint

לוחות שנה של SharePoint הם סוגי רשימות קלאסיים המבולבלים לעתים בלוחות השנה של קבוצת Microsoft 365.
 
כדי ליצור לוח שנה של SharePoint, בצע שלבים אלה:
 
1.  בחר את סמל גלגל השיניים **הגדרות** ולאחר מכן בחר **הוסף יישום**.
2.  בתיבת החיפוש חפש **יישום** , הקלד **לוח שנה** והקש enter.
3.  בחר את היישום שכותרתו ' **לוח שנה**' ולאחר מכן הזן שם עבור לוח השנה שלך.

לקבלת מידע נוסף אודות קבוצות Microsoft 365, ראה מידע [אודות קבוצות microsoft 365](https://support.office.com/article/Learn-about-Office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2).
