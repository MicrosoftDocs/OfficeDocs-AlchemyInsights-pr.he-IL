# <a name="add-a-sharepoint-calendar"></a>הוספת לוח שנה של SharePoint

לוחות שנה של SharePoint הם סוגי רשימה קלאסיים שלעתים מבולבלים עם Microsoft 365 לוחות שנה קבוצתיים.
 
כדי ליצור לוח שנה של SharePoint, בצע את הפעולות הבאות:
 
1.  בחר בסמל **הגדרות** הציוד ולאחר מכן בחר **בהוסף יישום**.
2.  בתיבה חפש **יישום** , הקלד **לוח שנה** והקש enter.
3.  בחר ביישום **לוח השנה**שכותרתו ולאחר מכן הזן שם עבור לוח השנה שלך.

לקבלת מידע נוסף על Microsoft 365 קבוצות, ראה [למד אודות הקבוצות של microsoft 365](https://support.office.com/article/Learn-about-Office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2).
