# <a name="email-forwarding-report"></a>דוח עבור העברת דואר אלקטרוני

כדי לעיין בדוח שמציג כמה הודעות שהועברו באופן אוטומטי קיימות בארגון שלך, ראה [תובנות לגבי זרימת דואר](https://docs.microsoft.com//office365/securitycompliance/mail-flow-insights-v2) ב[מרכז האבטחה והתאימות &amp;של Microsoft 365](https://protection.office.com/#/homepage).
  
שים לב: אם זוהי הפעם הראשונה שאתה נכנס למרכז התאימות והאבטחה, ייתכן שתצטרך לרענן את הדפדפן לאחר כמה דקות כדי לראות את כל האפשרויות הזמינות.
