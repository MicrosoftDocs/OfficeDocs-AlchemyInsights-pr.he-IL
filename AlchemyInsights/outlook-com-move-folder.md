# <a name="move-a-folder"></a>העברת תיקיה

באפשרותך לגרור תיקיות תחת **מועדפים** כדי לסדר אותם מחדש.
  
אין באפשרותך לסדר מחדש את התיקיות תחת **תיקיות** או להעביר תיקיית משנה כדי להפוך תיקיה.

*הערה:* אם תעביר תיקיה כדי להפוך את תיקיית משנה, צור תיקיה חדשה והעתק כל הדוא ל שלך מתוך תיקיית משנה לתיקיה החדשה.
  
לקבלת מידע נוסף, ראה [עבודה עם תיקיות של הודעות ב- Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a).
