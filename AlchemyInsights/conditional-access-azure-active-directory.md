# <a name="conditional-access"></a>גישה מותנית

גישה מותנית היא יכולת של תכלת Active Directory. באמצעות גישה מותנית, באפשרותך ליישם החלטות בקרת גישה אוטומטיות לגישה ליישומי הענן שלך המבוססים על תנאים. לקבלת מידע נוסף, בקר [בקביעת התצורה של מדיניות מבוססת מיקום](https://docs.microsoft.com/azure/active-directory/conditional-access/overview).

כמנהל IT, באפשרותך גם לשלוט בגישה למשאבי SharePoint ו-OneDrive בהתבסס על מיקומי רשת או מכשירים מוגדרים שאתה נותן בהם אמון. לקבלת פרטים נוספים, בקר בקישורים שלהלן.

- [שליטה בגישה אל נתוני SharePoint Online ו- OneDrive על בסיס מיקומי רשת](https://docs.microsoft.com/sharepoint/control-access-based-on-network-location)

- [בקרת גישה ממכשירים לא מנוהלים](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)
