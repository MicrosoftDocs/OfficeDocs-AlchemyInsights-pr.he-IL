# <a name="how-to-manage-add-ins-while-reading-a-message"></a>אופן הניהול של תוספות בעת קריאת הודעה

1. ב-Outlook באינטרנט, בחר הודעה.
    
2. בחר באפשרות **פעולות נוספות** בחלק העליון של ההודעה.

3. עבור אל החלק התחתון של הרשימה ובחר **קבל תוספות**.
    
4. בדף תוספות עבור Outlook, בחר את התוספת הרצויה.
    
    לקבלת תוספות ללא תשלום, בחר **הוסף** כדי להוסיף אותו לתיבת הדואר שלך.
    
    לקבלת תוספות בתשלום ומבחנים ללא תשלום, בחר באפשרות **קבל אותו** כדי להתחיל ברכישה.
    
לקבלת מידע נוסף, ראה [שימוש בתוספות ב-Outlook באינטרנט](https://support.microsoft.com/office/using-add-ins-in-outlook-on-the-web-8f2ce816-5df4-44a5-958c-f7f9d6dabdce).
