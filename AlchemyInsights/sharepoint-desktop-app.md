# <a name="desktop-app-for-sharepoint"></a>יישום בשולחן העבודה עבור SharePoint

הארגון שלך מוזן בתוכנית תצוגה מקדימה פרטית עבור יישום בשולחן העבודה עבור SharePoint.

Microsoft אינה תומכת כעת App שולחן העבודה עבור SharePoint (תצוגה מקדימה פרטית). נא השתמש השותפים שלך ופעולתם של תצוגה מקדימה פרטיים של Microsoft כדי לקבל סיוע App שולחן העבודה עבור SharePoint.
