# <a name="restoring-a-deleted-team-in-microsoft-teams"></a>שחזור צוות שנמחק ב-Microsoft Teams

האם בטעות מחקת צוות? אם עברו פחות מ-30 יום מאז שמחקת את הצוות, יש לך מזל; באפשרותך לשחזר אותו. כדי לעשות זאת, בצע את השלבים המתוארים כאן: [שחזור צוות שנמחק](https://docs.microsoft.com/microsoftteams/archive-or-delete-a-team#restore-a-deleted-team).
