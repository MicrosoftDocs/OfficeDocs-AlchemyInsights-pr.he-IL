# <a name="microsoft-stream-licensing-overview"></a>סקירה כללית של הרישוי של Microsoft Stream

Microsoft Stream הוא שירות ענן המורשה על בסיס מנוי לכל משתמש. הוא זמין כחלק ממנויים זכאים של Microsoft 365. לקבלת מידע על כל סוג מנוי, ראה Microsoft 365 תמחור.
