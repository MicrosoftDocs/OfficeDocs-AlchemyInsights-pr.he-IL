# <a name="read-receipts"></a>אישורי קריאה

לא ניתן לבקש אישור קריאה ב- Outlook.com, אך באפשרותך לבקש אישורי קריאה עבור הודעות Outlook.com שנשלחו באמצעות Outlook עבור Windows. כדי ללמוד כיצד לעשות זאת, ראה [הוספת ובקש לקרוא קבלות והודעות משלוח](https://support.office.com/article/a34bf70a-4c2c-4461-b2a1-12e4a7a92141).
  
כדי לבחור כיצד Outlook.com מגיב לבקשות לאישורי קריאה, עבור אל [ההגדרות של אישור קריאה](https://outlook.live.com/mail/options/mail/handling/readReceipts).
