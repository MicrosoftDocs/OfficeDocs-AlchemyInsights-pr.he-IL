# <a name="onedrive-is-stuck-on-processing-changes"></a>OneDrive תקוע על שינויים בעיבוד

אם OneDrive מציג את ההודעה "עיבוד שינויים" במשך זמן רב, ייתכן שהסיבה לכך היא שקובץ מקוון פתוח, הרבה קבצים תקועים בתור, או סינכרון קבצים גדול מאוד. קריאת  [OneDrive תקועה על "עיבוד שינויים"](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) לקבלת שלבים מוצעים ומידע נוסף.

אם אתה זקוק לעזרה נוספת, באפשרותך  [לשלוח בדואר אלקטרוני תמיכה ב-OneDrive](https://go.microsoft.com/fwlink/p/?LinkId=528676).
