# <a name="onedrive-is-stuck-on-processing-changes"></a>OneDrive תקוע בעיבוד שינויים

אם OneDrive מציג "עיבוד שינויים" במשך זמן רב, זה יכול להיות בגלל שיש לך קובץ מקוון פתוח, המון קבצים תקועים בתור, או קובץ גדול מאוד סינכרון. קרא [Onedrive תקוע ב"עיבוד שינויים"](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) עבור שלבים מוצעים ומידע נוסף.

אם אתה זקוק לעזרה נוספת, אתה יכול [לשלוח דוא ל OneDrive תמיכה](https://go.microsoft.com/fwlink/p/?LinkId=528676).
