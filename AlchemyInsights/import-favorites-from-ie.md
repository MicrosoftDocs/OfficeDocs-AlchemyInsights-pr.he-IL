# <a name="import-favorites-from-internet-explorer-to-microsoft-edge"></a>ייבוא מועדפים מ- Internet Explorer ל- Microsoft Edge

1. פתח את Microsoft Edge ולאחר מכן בחר **הגדרות ועוד ...**  >  **הגדרות**  >  **ייבוא נתוני דפדפן**.

1. בתפריט **ייבוא מ,** בחר **Microsoft Internet Explorer** ולאחר מכן בחר **ייבוא**.

1. **ב-All done!** תיבת הדו-שיח, בחר בוצע.

המועדפים המיובאים צריכים להיות בתיקיה, לדוגמה, **מיובאים** **או מועדפים אחרים,** **סרגל המועדפים.** באפשרותך להשאיר אותם במקום שבו הם נמצאים או להעביר אותם לתיקיה אחרת.
