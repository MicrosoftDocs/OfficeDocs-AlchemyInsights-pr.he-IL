# <a name="share-a-communication-site-with-external-users"></a>שיתוף אתר תקשורת עם משתמשים חיצוניים

כדי להפעיל שיתוף משתמשים חיצוני עבור אתרי תקשורת, בצע את הפעולות הבאות: 
  
1. פתח את **מרכז הניהול המודרני** -אם אתה במרכז הניהול הקלאסי תראה **לנסות את זה עכשיו** או * * לנסות את התצוגה המקדימה * * לחצן מימין העליון. זה ייקח אתכם למרכז המנהלה המודרני. 
  
2. בחר **אתרים פעילים.**
  
3. בחר אתר הכולל **תבנית אתר תקשורת**. 
  
4. בחרו בלחצן ' **שיתוף** ' מהתפריט העליון. 
  
 **הערה:** אם בחרת יותר מאתר אחד, לחצן **השיתוף** יופיע כלא זמין. בחר אתר אחד בלבד.
