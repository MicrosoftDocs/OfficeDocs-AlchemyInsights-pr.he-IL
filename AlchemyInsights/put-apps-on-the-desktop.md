# <a name="put-apps-on-the-desktop"></a>שים אפליקציות על שולחן העבודה

חפש את האפליקציה בתפריט **התחל** , ותפוס את סמל app על ידי החזקת לחצן העכבר השמאלי והזזת הסמל לשולחן העבודה.

בנוסף, באפשרותך להקיש על **מקש Windows + R** ולאחר מכן להקליד בפקודה זו:

`shell:appsfolder`

פעולה זו תפתח את תיקיית **היישומים** **בסייר הקבצים**. עכשיו אתה יכול לגרור ולשחרר את סמל app לשולחן העבודה.
