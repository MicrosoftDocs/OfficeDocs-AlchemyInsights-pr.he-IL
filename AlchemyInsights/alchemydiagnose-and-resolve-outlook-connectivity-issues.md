# <a name="diagnose-and-resolve-outlook-connectivity-issues"></a>לאבחן ולפתור בעיות קישוריות Outlook

כדי לאבחן ולתקן בעיות הקישוריות של Outlook שלך, באפשרותך להפעיל את [כלי הקישוריות של Outlook](https://aka.ms/SaRA-OutlookDisconnect-Alchemy) במחשב המושפע.
  
כלי זה לבצע בדיקות אבחון ולהציע פתרונות אפשריים עבורך לטפל בבעיה הקישוריות של Outlook.
