# <a name="getting-an-exchange-hybrid-key"></a>קבלת מפתח היברידית של Exchange

כדי להתחבר לארגון מקומי Exchange Online ב- Office 365, עליך להגדיר שרת Exchange אחד לפחות בארגון המקומית. שרת זה משמש כדי ליצור פריסה היברידית המקשר בין ארגון Exchange המקומית ל- Exchange Online. כדי לקבל מפתח היברידית עבור שרת זה, בצע את השלבים ב [אשף הפצה מפתח המוצר היברידית של Exchange](http://aka.ms/hybridkey).
