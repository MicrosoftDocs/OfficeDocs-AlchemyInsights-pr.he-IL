# <a name="getting-an-exchange-hybrid-key"></a>מקבל מפתח היברידי של Exchange

כדי לחבר את הארגון המקומי שלך ל-Exchange Online ב-Microsoft 365, עליך להתקין לפחות שרת Exchange אחד בארגון המקומי. שרת זה משמש ליצירת פריסה היברידית המחברת את ארגון Exchange המקומי שלך ל-Exchange Online. כדי לקבל מפתח היברידי עבור שרת זה, בצע את השלבים [באשף הפצת מפתח המוצר של Exchange](https://aka.ms/hybridkey).
