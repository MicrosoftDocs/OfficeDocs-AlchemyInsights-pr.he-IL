# <a name="create-a-folder-in-outlookcom"></a>צור תיקיה ב- Outlook.com

1. בחלק התחתון של חלונית התיקיות, בחר את **התיקיה החדשה**. ייתכן שיהיה עליך לגלול למטה בחלונית התיקיות כדי לראות את לחצן **תיקיה חדשה** .
2. בתיבת הטקסט, הזן שם עבור התיקיה ולאחר מכן הקש Enter.

לקבלת מידע נוסף, ראה [יצירת תיקיה ב- Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
