# <a name="add-a-password-to-a-word-document"></a>הוספת סיסמה למסמך Word

1. עבור אל **קובץ** > **מידע** > **הגנה על** > **הצפנת מסמך עם סיסמה**.
2. הזן את הסיסמה שבה ברצונך להשתמש ולאחר מכן בחר **באפשרות אישור**.
3. הזן מחדש את הסיסמה ולאחר מכן בחר **באפשרות אישור**.
