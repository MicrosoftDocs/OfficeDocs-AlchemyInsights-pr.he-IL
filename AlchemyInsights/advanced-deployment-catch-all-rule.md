# <a name="deploy-microsoft-365-apps"></a>פריסת יישומי Microsoft 365

לפריסת יישומים של Microsoft 365 למשתמשי קצה מרובים באמצעות כלי הפריסה של Office, מנהל תצורת נקודות הקצה של Microsoft, כחלק מתמונת דיסק של מערכת הפעלה או בתרחיש מחשב משותף באמצעות Windows Server עם שירותי שולחן עבודה מרוחק (RDS) או שירותי מסופים, שקול להשתמש במדריך צעד אחר צעד ב- **Microsoft 365 apps**
  
כדי להתחיל, ודא שאתה מחובר כמנהל הדיירים או כמנהל הכללי, ולאחר מכן לחץ כאן: https://aka.ms/proplusguidance .
