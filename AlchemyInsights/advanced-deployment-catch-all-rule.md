# <a name="deploy-office-apps"></a>פריסת יישומי Office

לפריסת יישומי Office למשתמשי קצה מרובים באמצעות הכלי לפריסת Office, מנהל תצורת נקודות הקצה של Microsoft, כחלק מתמונת דיסק של מערכת הפעלה או בתרחיש מחשב משותף באמצעות Windows Server עם שירותי שולחן עבודה מרוחק (RDS) או שירותי מסופים, שקול להשתמש במדריך השלבים שלב אחר שלב ב- **Microsoft 365 apps**
  
כדי להתחיל, ודא שאתה מחובר כמנהל הדיירים או כמנהל הכללי, ולאחר מכן לחץ [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance)כאן:.
