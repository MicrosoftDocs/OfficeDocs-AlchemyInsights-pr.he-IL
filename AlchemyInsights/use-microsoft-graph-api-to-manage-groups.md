# <a name="use-microsoft-graph-api-to-manage-groups"></a>שימוש ב-Microsoft Graph API לניהול קבוצות

קבוצות הן אוספי משתמשים ומנהלים אחרים המשתפים גישה למשאבים ב-Microsoft services או ביישום שלך. Microsoft Graph מספק ממשקי Api שבהם באפשרותך להשתמש כדי ליצור ולנהל סוגים שונים של קבוצות ופונקציונליות קבוצה בהתאם לתרחיש שלך. כל הפעולות הקשורות לקבוצה ב-Microsoft Graph דורשות הסכמה למנהלי מערכת. לקבלת מידע נוסף, ראה [עבודה עם קבוצות ב-Microsoft Graph](https://docs.microsoft.com/graph/api/resources/groups-overview).
