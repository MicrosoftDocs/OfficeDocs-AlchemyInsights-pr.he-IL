# <a name="who-has-a-file-checked-out"></a>למי יש קובץ שהוצא?

כדי לראות מי מוצא קובץ ב-SharePoint, באפשרותך לבצע אחת מהפעולות הבאות:
  
1. רחף מעל **הוצא** כדי לראות תיאור כלי שיידע אותך מי הוציא את הקובץ מאחסון. 
    
2. הוסף את העמודה ' **הוצא לפי** ' לתצוגת ברירת המחדל של הספריה. בדרך זו, כולם יכולים לראות בקלות מי הוציא מסמך. 
    
כדי ללמוד כיצד להוסיף עמודה שהוצאה על-ידי ב-SharePoint, ראה [יצירת עמודה ברשימה או בספריה](https://go.microsoft.com/fwlink/?linkid=2019591). 
  
ראה שאלות נפוצות יותר אודות הצ-אין או הוצאת החשבון, ראה [שאלות מובילות לגבי הוצאת קבצים,](https://go.microsoft.com/fwlink/?linkid=2018786)היכנס.
