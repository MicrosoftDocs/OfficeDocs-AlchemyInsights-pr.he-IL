# <a name="in-app-support-in-outlook-for-windows"></a>תמיכה ב-app ב-Outlook עבור Windows

כמנהל הדיירים, באפשרותך לקבל תמיכה ב-app עבור בעיות ב-Outlook עבור Windows. ב-Outlook, בחר **ב'עזרה**  >  **אנשי קשר**של עזרה '. הזן את שאלתך, וחבר ב-Outlook עבור צוות התמיכה של Windows תשיב להודעתך.
