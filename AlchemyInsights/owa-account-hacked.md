# <a name="outlook-on-the-web-account-hacked"></a>Outlook בחשבון האינטרנט פרוצים

אם החשבון שלך נפרץ, בצע את ההוראות [כיצד לאבטח ולשחזר את פונקציית הדואר האלקטרוני לחשבון ולתיבת הדואר החשודה של Microsoft 365](https://docs.microsoft.com/microsoft-365/security/office-365-security/responding-to-a-compromised-email-account).

כדי ללמוד עוד אודות שלבים נוספים שבאפשרותך לנקוט כדי למנוע את הסכנה של חשבונך, ראה [תגובה לחשבון דואר אלקטרוני שנפרץ ב-Microsoft 365](https://docs.microsoft.com/microsoft-365/security/office-365-security/responding-to-a-compromised-email-account).
