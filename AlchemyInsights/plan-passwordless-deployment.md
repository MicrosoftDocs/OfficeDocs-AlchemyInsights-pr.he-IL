# <a name="plan-your-passwordless-deployment"></a>תכנון הפריסה ללא סיסמה

שדרג ל כניסה חלופית באמצעות שיטת אימות ללא סיסמה. אימות דו-גורמי, כניסה לטלפון או מחוללי קוד יכולים לעזור להגביר את האבטחה ולהקשה על אחרים לפרוץ לחשבון שלך. 

בחר מתוך Windows Hello for Business, יישום האימות של Microsoft או מפתחות האבטחה של FIDO2. כדי לקבוע איזו שיטה היא הטובה ביותר ואופן הפריסה שלהם במכשיר שלך, השתמש במדריך [הפריסה ללא סיסמה](https://admin.microsoft.com/adminportal/home?#/modernonboarding/passwordlesssetup).
