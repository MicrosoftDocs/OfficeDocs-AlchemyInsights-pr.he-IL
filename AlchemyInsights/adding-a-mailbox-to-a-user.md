# <a name="adding-a-mailbox-to-a-user"></a>הוספת תיבת דואר למשתמש

כדי להוסיף תיבת דואר למשתמש שכבר נוצר או למשתמש חדש, יש להקצות למשתמש רשיון עבור Exchange Online. אם המשתמש כבר מוקצה רשיונות, בחר ב-Exchange Online תחת המקטע Apps בעמוד הרשיון כדי להתחיל את התהליך כדי ליצור תיבת דואר עבור המשתמש. לקבלת מידע נוסף, ראה [הקצאת רשיונות למשתמשים](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users).
