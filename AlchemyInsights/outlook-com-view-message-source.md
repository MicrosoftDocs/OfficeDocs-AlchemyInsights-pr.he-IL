# <a name="how-to-view-message-headers-or-message-source"></a>כיצד להציג כותרות של הודעות או מקור ההודעה

כדי לראות את הכותרות ופרטים אחרים בדואר אלקטרוני, בחר את שלוש נקודות בפינה העליונה של חלונית הקריאה ובחר באפשרות **הצג מקור ההודעה**.
