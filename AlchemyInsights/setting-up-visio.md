# <a name="setting-up-visio"></a>הגדרת Visio

נראה שאתה זקוק לעזרה בהגדרת Visio באמצעות מנוי Microsoft 365 שלך.
  
הודעה עבור משתמשי Mac: מוצר Visio אינו זמין עבור MacOS בשלב זה. אם יש לך מחשב, אתה אמור להיות מסוגל להתקין את התוכנה באמצעות השלבים שלהלן.
  
באפשרותך לבצע את השלבים המפורטים ב[מאמר זה](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710.aspx) כדי להשלים את תהליך ההגדרה.
