# <a name="cant-delete-files-or-photos-from-files"></a>אין אפשרות למחוק קבצים או תמונות מקבצים

ב- Outlook.com, **קבצים** מציג קבצים ותמונות המצורפים להודעות בתיבת הדואר שלך. כדי להסיר קבצים ותמונות מקבצים, עליך למחוק הודעות דואר אלקטרוני שהם מצורפים. לאחר מחיקת ההודעות, עליך גם למחוק אותם מהתיקיה פריטים שנמחקו.

לקבלת מידע נוסף, קרא [כיצד למחוק קבצים או תמונות ב- Outlook.com](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d?wt.mc_id=Office_Outlook_com_Alchemy).
