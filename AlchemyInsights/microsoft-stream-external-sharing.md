# <a name="microsoft-stream-external-sharing"></a>שיתוף חיצוני ב-Microsoft Stream

Microsoft Stream עדיין אינו תומך בתרחישי שיתוף חיצוניים עבור אורחים או אנשים מחוץ לארגון שלך. אנו יודעים שתכונות אלה דרושות מאוד, ואנו עובדים על תוכניות לגבי האופן שבו ניתן לבנות תכונות אלה.

- [שיתוף חיצוני בזרם](https://docs.microsoft.com/stream/portal-share-video#external-sharing)

- [שיתוף סרטוני הווידאו שלך ב-Microsoft Stream](https://docs.microsoft.com/stream/portal-share-video)
