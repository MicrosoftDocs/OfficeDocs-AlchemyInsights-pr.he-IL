# <a name="restore-deleted-files-or-folders-in-onedrive"></a>שחזור קבצים או תיקיות שנמחקו ב-OneDrive

אם מחקת בטעות קובץ או תיקיה ב-OneDrive, ייתכן שתוכל לשחזר אותו מאוחר יותר מסל המיחזור OneDrive.

* [שחזור קבצים או תיקיות שנמחקו ב-OneDrive](https://support.office.com/article/restore-deleted-files-or-folders-in-onedrive-949ada80-0026-4db3-a953-c99083e6a84f)

אם המון קבצים OneDrive שלך למחוק, הוחלף, פגום, או נגוע בתוכנות זדוניות, אתה יכול לשחזר את כל OneDrive לפעם הקודמת.

* [שחזר את ה-OneDrive שלך](https://support.office.com/article/Restore-your-OneDrive-fa231298-759d-41cf-bcd0-25ac53eb8a15)
