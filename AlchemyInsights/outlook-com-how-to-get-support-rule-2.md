# <a name="how-to-get-support-in-outlookcom"></a>כיצד ניתן לקבל תמיכה ב-Outlook.com

הזן את השאלה שלך או תאר את הבעיה שלך בתיבה **עזרה** לעיל ולאחר מכן בחר **קבל עזרה**.
