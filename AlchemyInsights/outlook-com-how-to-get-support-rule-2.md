# <a name="how-to-get-support-in-outlookcom"></a>כיצד לקבל תמיכה ב- Outlook.com

מצטערים, Outlook.com אינה מציעה תמיכה טלפונית, אך יש לנו צוותים ייעודיים היכולים לסייע לך.
הזן את השאלה או לתאר את הבעיה בתיבה **לסייע** לעיל, ולאחר מכן בחר **לקבל עזרה**.
