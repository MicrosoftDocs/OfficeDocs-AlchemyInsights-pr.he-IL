# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>שדרג לגירסה המלאה של Microsoft צוותים

באפשרותך לשדרג מ- Microsoft צוותים (ללא תשלום) לגירסה המלאה של Microsoft צוותים בכל פעם שאתה מוכן.
  
הדרך הטובה ביותר כדי להתחיל את תהליך השדרוג היא ללחוץ על **הודע לנו להתקשר אליך** להלן. 
  
סוכן התמיכה ייצור קשר עם עזרה בחר תוכנית Office 365, הזן את מידע איש הקשר ותשלום, בחר את שם התחום הארגוני שלך ולאחר שדרוג לוח תורנויות של החבר שלך ואתה.
