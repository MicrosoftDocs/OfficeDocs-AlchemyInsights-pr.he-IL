# <a name="calendar-freebusy-issues-in-hybrid-environments"></a>בעיות הפנויים/לא פנויים בלוח השנה בסביבות היברידית

אם משתמשים נתקלים בבעיות לראות לוח שנה בסביבות היברידית מידע לגבי מועדים פנויים /, הפעל מחדש את אשף קביעת התצורה של היברידית. לקבלת מידע נוסף אודות אשף קביעת התצורה של היברידית, ראה את [אשף קביעת התצורה של היברידית](https://go.microsoft.com/fwlink/p/?linkid=528149).
  
עבור שלבים בסביבות היברידית לפתרון בעיות, ראה [פתרון בעיות פריסה היברידית](https://technet.microsoft.com/library/jj659053.aspx).
