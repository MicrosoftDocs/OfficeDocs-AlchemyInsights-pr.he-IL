# <a name="what-is-autosave"></a>מהי שמירה אוטומטית?

שמירה אוטומטית היא תכונה הזמינה ב-Excel, Word ו-PowerPoint עבור מנויי Microsoft 365 שמצילים את הקובץ שלך באופן אוטומטי, כל כמה שניות, בזמן שאתה עובד. 

לשאלות נפוצות, ראה [מהי שמירה אוטומטית?](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
