# <a name="change-your-profile-name"></a>לשנות את שם הפרופיל שלך

1. מעבר אל [מידע חשבון Microsoft שלך](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. בחר **לערוך שם**. 
    
3. הזן שם פרטי ושם משפחה, ובחר באפשרות **שמור**. 
    
לתשומת לבך, עד 24 שעות עבור שינויים שם הפרופיל שלך ייכנסו לתוקף.
  
לקבלת מידע נוסף, ראה [שינוי שם Outlook.com או תמונות הפרופיל שלך](https://go.microsoft.com/fwlink/?linkid=873110).
