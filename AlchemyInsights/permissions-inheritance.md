# <a name="how-permissions-inheritance-works-in-sharepoint"></a>אופן הפעולה של הרשאות בירושה ב-SharePoint

כברירת מחדל, הרשאות ב-SharePoint עוברות בירושה מלמעלה למעלה בהירארכיה. כך שקובץ יירש את ההרשאות שלו מהתיקיה, אשר יורשת את ההרשאות שלו מהספריה, אשר יורשת את ההרשאות שלו מהאתר, אשר יורש את ההרשאות שלו מאוסף האתרים.
  
לקבלת מידע אודות הסרת הרשאות ייחודיות ושחזור ירושה, ראה [עריכה וניהול של הרשאות עבור רשימה או ספריה](https://go.microsoft.com/fwlink/?linkid=869946).
