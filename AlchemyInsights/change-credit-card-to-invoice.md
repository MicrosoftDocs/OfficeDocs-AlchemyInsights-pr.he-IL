# <a name="change-from-credit-card-or-bank-account-payments-to-invoice"></a>שינוי מתשלום בכרטיס אשראי או מחשבון בנק לחשבונית

באפשרותך לשנות כדי לשלם באמצעות חשבונית רק אם ההזמנה שלך היא מעל כמות מסוימת. אם אינך עומד בדרישה, תראה את ההודעה "כמות ההזמנה שלך אינה מספיק גדולה כדי לעמוד בדרישות עבור חשבונית עם תיל בדיקה". 

אם כמות ההזמנה שלך גבוהה מדי, ייתכן שתידרש בדיקת אשראי ואתה תראה את ההודעה "לא זכאי לשנות לחשבונית. ההזמנה שלך דורשת בדיקת אשראי." במקרה זה, שנה כדי לשלם באמצעות חשבונית באופן ידני. 

כדי להשלים את השינוי הידני, בצע את [השלבים המפורטים ב כיצד ניתן לשנות מתשלום בכרטיס אשראי לחשבונית?](https://docs.microsoft.com/alchemyinsights/how-do-i-change-from-credit-card-payments-to-invoice)
