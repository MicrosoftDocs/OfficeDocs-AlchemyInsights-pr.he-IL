# <a name="asms-test-solution"></a>פתרון בדיקה של ASMS

לקוחות התכלת עם הסכם הארגון תכלת (EA), הסכם הלקוח (MCA) או הסכם השותפים של Microsoft (MPA) עשויים להציג ולהוריד את התמחור שלהם בפורטל התכלת.
