# <a name="how-to-view-and-manage-apps"></a>כיצד להציג ולנהל יישומים

באפשרותך להציג את היישומים שלך במרכז הניהול של Microsoft 365. 

1. במרכז הניהול, עבור לדף ' **חיוב**  >  **[את המוצרים שלך](https://go.microsoft.com/fwlink/p/?linkid=842054)** '.

2. בדף ' **[המוצרים שלך](https://go.microsoft.com/fwlink/p/?linkid=842054)** ', בחר את הכרטיסיה ' **יישומים** '.
