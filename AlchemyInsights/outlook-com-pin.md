# <a name="change-your-pin"></a>שינוי ה-PIN שלך

Outlook.com אינו משתמש ב-PIN, אך ייתכן שתיכנס למחשב באמצעות PIN או Windows שלום. כדי לשנות את ה-PIN ב-Windows 10:

1. בחר את לחצן התחל או הקש על מקש Windows בלוח המקשים, ולאחר **מכן בחר** > **Accounts** > **אפשרויות כניסה**של חשבונות.
2. בחר את לחצן **השינוי** תחת **PIN**.
3. הזן את ה-PIN הנוכחי שלך ולאחר מכן הזן ואשר את ה-PIN החדש שלך מתחת.

[קרא עוד אודות Windows שלום.](https://support.microsoft.com/help/17215/)
