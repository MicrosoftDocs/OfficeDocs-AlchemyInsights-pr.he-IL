# <a name="learn-about-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>קבלת מידע אודות מיפוי בין מדיניות מדור קודם של Microsoft Edge ומדיניות של Microsoft Edge

מדיניות הקבוצה של Microsoft Edge Legacy (גירסאות 45 וגירסאות קודמות) והגדרות ניהול מכשירים ניידים (MDM) ממפות את המדיניות הרלוונטית של Microsoft Edge גירסה 80. לקבלת מידע נוסף, ראה [מיפוי קצה של Microsoft edge למיפוי מדיניות של Microsoft edge](https://go.microsoft.com/fwlink/?linkid=2141665).
