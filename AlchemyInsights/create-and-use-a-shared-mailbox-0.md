# <a name="create-and-use-a-shared-mailbox"></a>יצירה ושימוש בתיבת דואר משותפת

תיבת דואר משותפת מאפשרת ליותר מאדם אחד לשלוח ולקבל דואר אלקטרוני עבור כינוי, כגון **sales@contoso.com**, מבלי להקצות רשיון נוסף.
  
[יצירת תיבת דואר משותפת מסבירה](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) כיצד: 
  
- יצירת תיבת דואר משותפת
    
- תן למשתמשים שלך הרשאות להשתמש בתיבת דואר משותפת
    
- גישה לתיבת הדואר המשותפת ב-Outlook או ב-Outlook באינטרנט
