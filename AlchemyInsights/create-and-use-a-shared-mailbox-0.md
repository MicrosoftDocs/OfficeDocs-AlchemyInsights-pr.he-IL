# <a name="create-and-use-a-shared-mailbox"></a>יצירה ושימוש של תיבת דואר משותפת

תיבת דואר משותפת מאפשרת ליותר מאדם אחד שליחה וקבלה של דואר אלקטרוני עבור כינוי, כמו **sales@contoso.com**, מבלי להקצות רשיון נוסף.
  
[צור תיבת דואר משותפת](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) מסביר כיצד: 
  
- יצירת תיבת דואר משותפת
    
- להעניק למשתמשים הרשאות לשימוש תיבת דואר משותפת
    
- Access תיבת הדואר המשותפת ב- Outlook או ב- Outlook באינטרנט
