# <a name="map-a-sharepoint-library-to-a-network-drive"></a>מיפוי ספריית SharePoint לכונן רשת

מיפוי ספריית ככונן רשת הוא זמני נתמכים רק באמצעות Internet Explorer. עליך מדי פעם לפתוח את אתר SharePoint ב- Internet Explorer ולבחור "הישאר מחובר" כדי למנוע את הפעלת תאריך תפוגה. במקום זאת, [לסנכרן קבצים SharePoint עם לקוח הסינכרון החדש של OneDrive](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88) </a> המספק [קבצים לפי דרישה](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e). Access את כל הקבצים שלך OneDrive ללא שימוש בשטח אחסון מקומי.

אם תבחר למפות כונן במקום[באמצעות לקוח סינכרון OneDrive חדש](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88), ודא שבצע את הפעולות המתוארות במאמר שלהלן. 


**כיצד להגדיר תצורה ולפתור את כונני הרשת הממופים**


- [ולקבוע את תצורת פתרון כונני הרשת הממופים](https://support.office.com/article/troubleshoot-mapped-network-drives-that-connect-to-sharepoint-online-ef399c67-4578-4c3a-adbe-0b489084eabe?ui=en-US&amp;rs=en-US&amp;ad=US).

הערה: עבור Internet Explorer 10 עם Windows 8 או Windows 7 מקבל "Access נדחתה" או "נתיב אינו נגיש" בעת מיפוי כונן, התקנת תיקון חם זה כדי לפתור בעיה זו. עבור אל [השגיאה בעת פתיחת ספריית מסמכים של SharePoint ב- Windows Explorer או למפות כונן רשת לספריית לאחר התקנת Internet Explorer 10](https://support.microsoft.com/help/2846960).
