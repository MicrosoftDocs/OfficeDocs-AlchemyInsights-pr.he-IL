# <a name="map-a-sharepoint-library-to-a-network-drive"></a>מיפוי ספריית SharePoint לכונן רשת

מיפוי ספריה ככונן רשת הוא זמני ונתמך רק באמצעות Internet Explorer. לעתים עליך לפתוח את אתר SharePoint ב-Internet Explorer ולבחור **באפשרות השאר מחובר** כדי למנוע מההפעלה לפוג. במקום זאת, [סנכרן קבצי SharePoint עם לקוח הסינכרון החדש של OneDrive](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88) </a> המספק [קבצים לפי דרישה](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e). גש אל כל הקבצים שלך ב-OneDrive ללא שימוש בשטח אחסון מקומי.

אם תבחר למפות כונן במקום [להשתמש בלקוח הסינכרון החדש של OneDrive](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88), ודא שאתה מבצע את השלבים במאמר שלהלן. 


**כיצד להגדיר ולפתור בעיות בכונני רשת ממופים**


ראה [פתרון בעיות בכונני רשת ממופים המתחברים ל-SharePoint online](https://docs.microsoft.com/sharepoint/support/administration/troubleshoot-mapped-network-drives).

הערה: אם אתה משתמש ב-Internet Explorer 10 עם Windows 8 או Windows 7, וקבלת **גישה שנדחתה** או **שהנתיב אינו נגיש** בעת מיפוי כונן, התקן [תיקון חם זה](https://support.microsoft.com/help/2846960) כדי לפתור בעיה זו.
