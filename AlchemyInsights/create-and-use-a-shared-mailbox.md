# <a name="create-and-use-a-shared-mailbox"></a>יצירה ושימוש של תיבת דואר משותפת

תיבת דואר משותפת מאפשרת ליותר מאדם אחד שליחה וקבלה של דואר אלקטרוני עבור כינוי, כמו sales@contoso.com, מבלי להקצות רשיון נוסף.
  
- [בצע את השלבים הבאים](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) במרכז admin Microsoft 365 כדי להתחיל. 

באפשרותך גם לקרוא את המאמר [יצירת תיבת דואר משותפת](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9.aspx) , תדווח לך כל מה שעליך לדעת אודות תיבות דואר משותפות, כולל כיצד להשתמש בהם.
