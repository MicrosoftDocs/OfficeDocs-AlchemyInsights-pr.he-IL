כדי לקבל את המערך המלא של התכונות של Microsoft Edge לרשותכם, עליך להתחבר לאינטרנט. בנוסף, כדי לאפשר לדפדפן לקיים תקשורת באמצעות חומות אש ומנגנוני אבטחה אחרים באינטרנט, עליך להוסיף ערכה של כתובות Url של תחומים [לרשימת נקודות הקצה המותרת](https://go.microsoft.com/fwlink/?linkid=2135054).
