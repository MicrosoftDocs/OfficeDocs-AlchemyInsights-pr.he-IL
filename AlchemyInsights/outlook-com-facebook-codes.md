# <a name="not-receiving-facebook-codes-using-outlookcom"></a>לא מקבל קודים של Facebook באמצעות Outlook.com

אם ביקשת סיסמה חדשה מ-Facebook, אך לא קיבלת את הודעת הדואר האלקטרוני לאיפוס הסיסמה:

- בדוק את [התיקיה ' דואר זבל ' או ' דואר זבל '](https://outlook.live.com/mail/junkemail).
- נסה [שוב לאפס את הסיסמה שלך](https://aka.ms/facebook-password-reset)ב-Facebook.

אם אין באפשרותך לאפס את הסיסמה מכיוון שאין באפשרותך לגשת לדואר האלקטרוני או למספר הטלפון הנייד בחשבון שלך, נסה את [העצות לאיפוס סיסמה של Facebook](https://aka.ms/facebook-password-help).
