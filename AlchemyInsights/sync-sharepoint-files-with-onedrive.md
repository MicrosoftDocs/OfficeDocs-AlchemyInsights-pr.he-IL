# <a name="sync-sharepoint-files-with-the-new-onedrive-sync-client"></a>סינכרון קבצי SharePoint באמצעות לקוח הסינכרון החדש של OneDrive

<p>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">הפקודה 'פתח באמצעות הסייר' פותחת מופע מקומי של סייר Windows שמציג את מבנה התיקיות בשרת שמארח את אתר SharePoint. למרות זאת, מומלץ <a href="https://support.office.com/he-IL/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88">לסנכרן קבצי SharePoint באמצעות לקוח הסינכרון החדש של OneDrive</a> שמספק את התכונה <a href="https://support.office.com/he-IL/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e">קבצים לפי דרישה</a>, מכיוון שהוא מספק גישה מקומית לקבצים שלך ומציע את הביצועים הטובים ביותר</span></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">אם בחרת להשתמש בתצוגת הסייר במקום להשתמש בלקוח הסינכרון החדש, הקפד לפעול בהתאם לשלבים ולשיטות העבודה המומלצות במאמרים שלהלן.</span></p> <ul> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/he-IL/article/How-to-use-the-Open-with-Explorer-command-to-troubleshoot-issues-in-SharePoint-Online-87155331-0c92-4224-a4c1-da5c21c4ade4">כיצד ניתן להשתמש בפקודה 'פתח באמצעות הסייר' כדי לפתור בעיות ב- SharePoint Online</a></span></li> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/he-IL/article/copy-or-move-library-files-by-using-open-with-explorer-aaee7bfb-e2a1-42ee-8fc0-bcc0754f04d2">העתקה או העברה של קבצי ספריה באמצעות הפקודה 'פתח באמצעות הסייר'</a></span></li> </ul> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">הערה:</span></u></strong><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> לחצן 'פתח באמצעות הסייר' אינו מופיע בחוויית הספריה החדשה. לחץ על התפריט הנפתח <strong>תצוגה</strong> בפינה השמאלית העליונה (שם התפריט הנפתח משתנה בהתאם לתצוגה הנוכחית) ולאחר מכן לחץ על <strong>הצג</strong> בסייר הקבצים.</span></p> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">הערה:</span></u></strong><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> ב- SharePoint, הפקודה 'פתח באמצעות הסייר' משתמשת ב<strong>פקדי ActiveX</strong>, ולכן היא נתמכת רק ב- Internet Explorer 10 או ב- Internet Explorer 11. הפקודה 'פתח באמצעות הסייר' אינה פועלת ב- Windows עם Microsoft Edge,‏ Google Chrome או Mozilla Firefox ואינה פועלת בפלטפורמת Mac. כתוצאה מכך, ייתכן שהאפשרות &ldquo;תצוגת סייר&rdquo; תופיע באפור.</span></p> <ul> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/he-IL/article/Why-SharePoint-ribbon-buttons-are-unavailable-48b0939a-2efb-4e79-b5e8-b2c4cb5d04ca">מדוע לחצני רצועת הכלים של SharePoint אינם זמינים או מופיעים באפור.</a></span></li> </ul> <p>&nbsp;</p>
