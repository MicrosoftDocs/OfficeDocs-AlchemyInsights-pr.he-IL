# <a name="why-cant-i-edit-this-file"></a>מדוע אין באפשרותי לערוך קובץ זה?

אם מסמך, גליון עבודה או מצגת נעולים לעריכה ולא ניתן לערוך אותם ביישומי שולחן העבודה של Windows, ייתכן שהקובץ נערך על-ידי מישהו אחר, או שהקובץ הוצא.

מבט מהיר על חלק מהסיבות [שבגללן ייתכן שלא תוכל לערוך את הקובץ](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).
