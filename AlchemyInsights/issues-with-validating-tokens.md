# <a name="issues-with-validating-tokens"></a>בעיות באימות אסימונים

כדי לפתור בעיות בנושא אימות אסימונים, ראה:

- [אימות אסימונים](https://docs.microsoft.com/azure/active-directory/develop/access-tokens#validating-tokens) -לקבלת מידע על אימות אסימונים של access.

- [אמת את אסימון המזהה](https://docs.microsoft.com/azure/active-directory/develop/v2-protocols-oidc#validate-the-id-token) -לקבלת מידע על אימות אסימונים של מזהה.
