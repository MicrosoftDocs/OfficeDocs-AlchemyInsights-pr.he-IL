# <a name="create-a-new-folder-in-outlookcom"></a>צור תיקיה חדשה ב- Outlook.com

1. תחת **תיקיות** בחלונית הניווט, בחר את **התיקיה החדשה**.

2. הקלד שם עבור התיקיה ולאחר מכן הקש Enter.

לקבלת מידע נוסף, ראה [יצירת תיקיה ב- Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a?wt.mc_id=Office_Outlook_com_Alchemy).
