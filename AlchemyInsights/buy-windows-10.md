# <a name="how-to-buy-a-copy-of-windows-10"></a>כיצד לקנות עותק של Windows 10

לקבלת תמחור עדכני ואופן ה קניה והורדה של גירסה מלאה של Windows 10, עבור אל [Microsoft Store](https://www.microsoft.com/store/b/windows).

**הערה:** השדרוג Windows 10 ללא תשלום דרך יישום קבל את Windows 10 (GWX) הסתיים ב- 29 ביולי 2016.

**חשוב:** אם אתה משתמש ב- Windows 7, Microsoft ממליצה לעבור למחשב חדש שפועל בו Windows 10, כאשר Windows 7 הגיע לסוף מחזור חיי התמיכה שלו ב- 14 בינואר 2020.
