# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>הרשאות תיבת דואר שהוקצתה בסביבה היברידית

בהתאם לגירסת Exchange שהתקנת בארגון המקומי, ייתכן שיהיה עליך לנקוט בכמה צעדים נוספים כדי להגדיר הרשאות תיבת דואר מוקצות בין הארגון המקומי שלך ל-Exchange Online ב-Microsoft 365. לקבלת מידע נוסף אודות אופן ההגדרה של הרשאות שהוקצו, הסתכל על [קביעת תצורה של Exchange כדי לתמוך בהרשאות תיבת דואר שהוקצו בפריסה היברידית](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx).
