# <a name="get-help-with-windows-10-desktop"></a>קבל עזרה עבור Windows 10 Desktop

- כדי ליצור קיצור דרך בשולחן העבודה, ראה [הצמדת אפליקציות ותיקיות לשולחן העבודה או לשורת המשימות](https://support.microsoft.com/windows/pin-apps-and-folders-to-the-desktop-or-taskbar-f3c749fb-e298-4cf1-adda-7fd635df6bb0).

- כדי להפוך את הסמלים או הטקסט לגדולים יותר ב- Windows 10, [ראה שינוי הגודל של סמלי שולחן העבודה.](https://support.microsoft.com/windows/change-the-size-of-your-desktop-icons-85a9d341-2a4f-3d96-c796-ae116a187211)

- כדי להציג סמלי שולחן עבודה ב- Windows 10, ראה [הצגת סמלי שולחן עבודה ב- Windows 10.](https://support.microsoft.com/windows/show-desktop-icons-in-windows-10-c13270f0-3812-c71d-f27e-29aa32588b20)
