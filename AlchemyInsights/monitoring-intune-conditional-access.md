# <a name="monitoring-intune-conditional-access"></a>פיקוח על Access מותנה Intune

סקור את המשאבים המפורטים להלן כדי לפתור את הבעיה כעת.
  
1. להשתמש [בפורטל פתרון בעיות Intune](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) כדי לאבחן ולפתור שחלק מהכשלים נפוצות. עיין במסמך זה לקבלת פרטים נוספים. 
    
2. סקור [Intune TechNet ](https://social.technet.microsoft.com/forums/en-us/home?forum=microsoftintuneprod)כדי למצוא תשובות ופתרונות לבעיות נפוצות.
    
3. בדוק את [תקינות השירות](https://portal.office.com/AdminPortal/Home#/servicehealth) כדי לראות את המצב של השירות. 
    
4. סקירת [תיעוד Intune](https://docs.microsoft.com/en-us/intune/) עבור תכנון, מעשיים, פתרון בעיות מאמרים ועוד.
