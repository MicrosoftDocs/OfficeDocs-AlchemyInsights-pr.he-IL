# <a name="enable-seamless-sso"></a>אפשר SSO חלקה

תכלת הרקיע Active Directory (תכלת הרקיע AD) חלקה כניסה יחידה (SSO חלקה) באופן אוטומטי מכניסה למשתמשים כאשר הם בשולחנות העבודה שלהם ארגוניים המחוברים לרשת הארגונית שלך.
  
[בצע את הפעולות המתוארות במאמר זה כדי לאפשר SSO חלקה](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
