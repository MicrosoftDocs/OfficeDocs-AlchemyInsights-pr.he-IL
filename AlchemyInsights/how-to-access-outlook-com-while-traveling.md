# <a name="how-to-access-outlookcom-while-traveling"></a>כיצד לגשת Outlook.com במהלך הנסיעה

כדי לגשת ל-Outlook.com במהלך הנסיעה, ודא שפרטי האבטחה של חשבון Microsoft שלך-כתובת דואר אלקטרוני חלופית או מספר טלפון שבו ניתן לקבל קודי אבטחה-מעודכנים. לקבלת מידע נוסף, ראה [כיצד לגשת לOutlook.com בעת הנסיעה](https://support.office.com/article/c44f16da-7156-4890-853c-286aafeda87e?wt.mc_id=Office_Outlook_com_Alchemy).
