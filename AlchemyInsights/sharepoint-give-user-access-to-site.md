# <a name="give-users-access-to-sharepoint-site"></a>הענקת גישה למשתמשים לאתר SharePoint

התכונה ' בקשת גישה ' מאפשרת לאנשים לבקש גישה לתוכן שאינו כולל כעת הרשאה לראות. כבעלים של אתר, באפשרותך לקבוע את תצורת התכונה כך שתשלח לך דואר כאשר מישהו יבקש גישה לאתר. לקבלת מידע נוסף, בקר [בהגדרת בקשות גישה וניהולן](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3).

להלן כמה קישורים נפוצים לגבי בקשות גישה:

- [הגישה נדחתה ' כדי לגשת לרשימת בקשות ' או "בקשת אישור נכשלה" בעת עיבוד בקשה ממתינה ב-SharePoint Online](https://docs.microsoft.com/sharepoint/support/sharing-and-permissions/request-approval-failed)

- [שינוי ההגדרות של בקשות הגישה](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#bk_enableallow)

- [אישור או דחייה של בקשות גישה עבור אתר](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462558)

- [ניהול הזמנות למשתמשים חיצוניים (SharePoint Online בלבד)](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc334189260)

- [שליחה מחדש של הזמנה](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462560)
