# <a name="microphone-isnt-working-in-teams"></a>המיקרופון אינו פועל ב-Teams

התחל לפתור את הבעיה על-ידי ביצוע הפעולות הבאות:

- בדיקה אם קיימים עדכונים עבור Windows ו-Teams.
- סגירת יישומים אחרים המשתמשים במיקרופון.
- ביצוע שיחת בדיקה כדי לבדוק אם המיקרופון פועל כהלכה.

לקבלת פרטים ושלבים נוספים לפתרון [בעיות, ראה המיקרופון שלי אינו פועל ב-teams](https://support.microsoft.com/office/666d1123-9dd0-4a31-ad2e-a758b204f33a).
