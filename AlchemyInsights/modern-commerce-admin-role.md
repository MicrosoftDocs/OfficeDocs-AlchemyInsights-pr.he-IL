# <a name="modern-commerce-administrator-role"></a>תפקיד מנהל מסחר מודרני

תפקיד זה מוקצה באופן אוטומטי למשתמשים העושים רכישה כדי להעניק להם גישה מוגבלת לפורטל המנהלה כדי לנהל את הרכישה. משתמשים אלה אינם מנהלים ואין להם הרשאות מוגברות.
