# <a name="troubleshoot-issues-with-dep-enrollment-in-microsoft-intune"></a>פתרון בעיות עם הרשמה DEP ב- Microsoft Intune

סקור את המשאבים המפורטים להלן כדי לפתור את הבעיה כעת.
  
1. אם התקן ה-DEP אין אפשרות להירשם MFA (אימות מגורמים רבים) מופעלת, בטל MFA. כעת MFA אינו נתמך עבור הרשמה DEP

2. להשתמש [בפורטל פתרון בעיות Intune](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) כדי לאבחן ולפתור כשלים הרשמה נפוצות. סקירת [מסמך זה](https://docs.microsoft.com/intune/help-desk-operators) לקבלת פרטים נוספים.

3. סקירת מסמכים אלה לקבלת רשימה של שגיאות נפוצות המונעות הרשמה ופתרונות לכל: [מדריך פתרון בעיות](https://support.microsoft.com/help/4039809/troubleshooting-ios-device-enrollment-in-intune) ו- [doc פתרון בעיות](https://docs.microsoft.com/intune-classic/troubleshoot/troubleshoot-device-enrollment-in-intune)

4. [למד אודות תוכנית הרשמה של ההתקן](https://docs.microsoft.com/intune/device-enrollment-program-enroll-ios).
