# <a name="troubleshoot-issues-with-dep-enrollment-in-microsoft-intune"></a>פתרון בעיות בהרשמת DEP ב-Microsoft Intune

סקור את המשאבים המפורטים להלן כדי לפתור את הבעיה שלך כעת.
  
1. אם התקן DEP אינו מסוגל להירשם ומשרד הרישום (אימות רב-גורמי) מופעל, נא בטל את משרד הרישום. כיום אין תמיכה ב-משרד הרישום עבור הרשמת DEP

2. השתמש [בפורטל לפתרון בעיות Intune](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) כדי לאבחן ולפתור כשלים נפוצים בהרשמה. לפרטים נוספים, עיין [במסמך זה](https://docs.microsoft.com/intune/help-desk-operators) .

3. סקור מסמכים אלה לקבלת רשימה של שגיאות נפוצות המונעות הרשמה והחלטות לכל: [מדריך לפתרון בעיות](https://support.microsoft.com/help/4039809/troubleshooting-ios-device-enrollment-in-intune) [ופתרון בעיות doc](https://docs.microsoft.com/intune-classic/troubleshoot/troubleshoot-device-enrollment-in-intune)

4. [למד אודות תוכנית ההרשמה להתקן](https://docs.microsoft.com/intune/device-enrollment-program-enroll-ios).
