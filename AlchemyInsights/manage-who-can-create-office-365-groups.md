# <a name="manage-who-can-create-microsoft-365-groups"></a>נהל את מי שיכול ליצור קבוצות של Microsoft 365

כברירת מחדל, כל אדם בארגון שלך יכול ליצור קבוצות O365.
  
אם ברצונך לשנות זאת, באפשרותך ליצור קבוצת אבטחה עם PowerShell המגבילה את יצירת הקבוצה לקבוצת משנה של משתמשים.
  
בצע את ההוראות שלב אחר שלב: [לנהל מי יכול ליצור Microsoft 365 קבוצות](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
