# <a name="manage-who-can-create-microsoft-365-groups"></a>ניהול מי יכול ליצור קבוצות Microsoft 365

כברירת מחדל, כל אדם בארגון שלך יכול ליצור קבוצות O365.
  
אם ברצונך לשנות זאת, באפשרותך ליצור קבוצת אבטחה עם PowerShell המגבילה את יצירת הקבוצה לקבוצת משנה של משתמשים.
  
בצע את ההוראות המפורטות כאן: [ניהול מי יכול ליצור קבוצות Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
