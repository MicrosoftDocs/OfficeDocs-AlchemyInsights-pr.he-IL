# <a name="app-registration-permissions"></a>הרשאות לרישום יישומים

[הרשאות לרישום יישומים עבור תפקידים מותאמים אישית ב-תכלת Active directory](https://docs.microsoft.com/azure/active-directory/roles/custom-available-permissions): מאמר זה מכיל את הרשאות רישום היישומים הזמינות הנוכחיות עבור הגדרות תפקידים מותאמות אישית ב-תכלת active Directory (תכלת לספירה).
