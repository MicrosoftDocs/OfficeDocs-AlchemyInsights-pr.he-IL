# <a name="removing-the-last-on-premises-exchange-server"></a>הסרת שרת Exchange המקומי האחרון

לפני הסרת שרת Exchange האחרון מהארגון המקומי, עליך לוודא שאתה מבין את ההשלכות של פעולה זו כדי לסייע לך להחליט אם אתה זקוק לשרת האחרון למטרות אחרות. לקבלת מידע נוסף, בדוק [כיצד ומתי לבטל את הדחיסה של שרתי Exchange המקומיים בפריסה היברידית](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
