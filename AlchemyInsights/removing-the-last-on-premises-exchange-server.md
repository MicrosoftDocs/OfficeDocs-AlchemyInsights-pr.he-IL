# <a name="removing-the-last-on-premises-exchange-server"></a>הסרת שרת Exchange המקומית האחרון

לפני הסרת שרת ה-Exchange האחרון מהארגון המקומית שלך, עליך לוודא שאתה מבין ההשלכות של פעולה זו כדי לסייע לך להחליט אם עליך זה השרת האחרון למטרות אחרות. לקבלת מידע נוסף, להוציא [כיצד ומתי הוצאת ממצב פעיל שרתי Exchange המקומית שלך בפריסה היברידית](https://technet.microsoft.com/en-us/library/dn931280%28v=exchg.150%29.aspx).
