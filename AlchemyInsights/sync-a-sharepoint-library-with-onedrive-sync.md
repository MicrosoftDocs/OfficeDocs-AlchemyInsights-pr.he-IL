# <a name="sync-a-sharepoint-library-with-onedrive-sync"></a>סינכרון ספריית SharePoint עם סינכרון OneDrive

כדי שקבצי SharePoint שלך יסונכרנו עם הכונן הקשיח המקומי שלך, בצע את השלבים הבאים:

1. נווט אל ספריית המסמכים ש רוצה לסנכרן עם הכונן הקשיח המקומי.
2. בחר בלחצן סנכרן מתוך אפשרויות התפריט.
3. בצע את השלב אחר שלב כדי להשלים את הסינכרון.
