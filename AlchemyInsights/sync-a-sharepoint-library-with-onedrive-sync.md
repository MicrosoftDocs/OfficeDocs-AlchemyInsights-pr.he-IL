# <a name="sync-a-sharepoint-library-with-onedrive-sync"></a>סינכרון ספריית SharePoint עם סינכרון OneDrive

כדי שקבצי SharePoint שלך יסתנכרנו לכונן הקשיח המקומי, השלם את השלבים הבאים:

1. נווט אל ספריית המסמכים שברצונך לסנכרן לכונן הקשיח המקומי.
2. בחרו בלחצן ' סנכרון ' מאפשרויות התפריט.
3. בצע את השלבים שלב אחר שלב כדי להשלים את הסינכרון.
