# <a name="import-contacts"></a>ייבוא אנשי קשר

לפני ייבוא אנשי קשר ל- Outlook באינטרנט, יהיה עליך לייצא אותם מתוך ספק דואר אלקטרוני אחרים שלך כקובץ csv. לאחר מכן, בצע את ההוראות שלהלן.

1. בחלק התחתון של הדף, בחר בסמל אנשים כדי לעבור אל הדף ' אנשים.
2. בסרגל הכלים, בחר **ניהול** > **ייבוא אנשי קשר**.
3. בחר **עיון**, בחר את קובץ ה-CSV, ולאחר מכן בחר **פתוח**.
4. בחר באפשרות **יבא**.

לקבלת מידע נוסף, ראה [ייבוא אנשי קשר ל- Outlook](https://support.office.com/article/bb796340-b58a-46c1-90c7-b549b8f3c5f8#ID0EAACAAA=Outlook_on_the_web).
