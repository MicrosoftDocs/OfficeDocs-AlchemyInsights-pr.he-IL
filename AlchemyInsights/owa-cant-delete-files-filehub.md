# <a name="cant-delete-files-or-photos-from-files-in-outlook-on-the-web"></a>אין אפשרות למחוק קבצים או תמונות מקבצים ב-Outlook באינטרנט

**קבצים** מציג את הקבצים והתמונות המצורפים להודעות בתיבת הדואר שלך. כדי להסיר קבצים ותמונות מקבצים, עליך למחוק את הודעות הדואר האלקטרוני שהן מצורפות להן. לאחר מחיקת ההודעות, עליך גם למחוק אותם מהתיקיה ' פריטים שנמחקו '.
