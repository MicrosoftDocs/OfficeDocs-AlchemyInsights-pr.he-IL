# <a name="cant-delete-files-or-photos-from-files-in-outlook-on-the-web"></a>אין אפשרות למחוק קבצים או תמונות מתוך קבצים ב- Outlook באינטרנט

**קבצים** מציג קבצים ותמונות המצורפים להודעות בתיבת הדואר שלך. כדי להסיר קבצים ותמונות מקבצים, עליך למחוק הודעות דואר אלקטרוני שהם מצורפים. לאחר מחיקת ההודעות, עליך גם למחוק אותם מהתיקיה פריטים שנמחקו.
