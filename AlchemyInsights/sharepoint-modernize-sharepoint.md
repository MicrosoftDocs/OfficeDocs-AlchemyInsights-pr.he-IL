# <a name="modernize-your-classic-sharepoint-experience"></a>מודרניזציה חוויית ה-SharePoint הקלאסית שלך

כדי להפוך את המתג לממשק המשתמש המודרני, סקור את המשאבים הבאים:

- [צור **אתרים מודרניים** ](https://support.office.com/article/create-a-team-site-in-sharepoint-ef10c1e7-15f3-42a3-98aa-b5972711777d) (אתר צוות או אתר תקשורת). השתמש [בספר המראה של SharePoint](https://lookbook.microsoft.com/assets/SharePoint_lookbook_2019.pdf) כדי לבנות רשת אינטרא-נט יפה ורבת עוצמה המחברת את מקום העבודה. ראה [דוגמאות](https://lookbook.microsoft.com/) וקבל [הדרכה לעיצוב](https://spdesign.azurewebsites.net/).
- [חבר את האתר שלך **לקבוצת microsoft 365** ](https://docs.microsoft.com/sharepoint/dev/transform/modernize-connect-to-office365-group) כדי להגביר את שיתוף הפעולה באמצעות Microsoft 365.
- המרה לחוויית **הרשימה והספריה המודרנית** על-ידי [הפעלת רשימות וספריות להצגה בממשק המשתמש המודרני](https://docs.microsoft.com/sharepoint/dev/transform/modernize-userinterface-lists-and-libraries).
- שינוי **הצורה** של **דפי אתר** מתוך **wiki**, **web part**ודפי בית קלאסיים [לדפים מודרניים בצד הלקוח](https://docs.microsoft.com/sharepoint/dev/transform/modernize-userinterface-site-pages). [סורק המודרניזציה של SharePoint](https://docs.microsoft.com/sharepoint/dev/transform/modernize-scanner) מספק מידע אודות שימוש בדפים אודות דפי ה-wiki והדפים הנוכחיים של האתר כדי לסייע לך לתעדף את המרת העמוד.

מידע נוסף:

- עבור סקירה צעד אחר צעד של מודרניזציה חוויית SharePoint הקלאסי שלך מודרני, לראות [מודרניזציה חוויית sharepoint הקלאסי שלך](https://docs.microsoft.com/sharepoint/dev/transform/modernize-classic-sites).
- ראו מדריך [לחוויה מודרנית](https://docs.microsoft.com/sharepoint/guide-to-sharepoint-modern-experience).
- ראה [התנסויות קלאסיות ומודרניות של SharePoint](https://support.office.com/article/sharepoint-classic-and-modern-experiences-5725c103-505d-4a6e-9350-300d3ec7d73f).
