# <a name="fix-authentication-failed-errors-in-microsoft-flow"></a>תיקון שגיאות "אימות שנכשלו" ב-Microsoft Flow

אם מתקבלת הודעת שגיאה "אימות נכשל" כאשר אתה מנסה להיכנס ל-Microsoft Flow, הסיבה לכך היא ששירות זרימה אינו זמין בתיקיה ' פעיל '. לקבלת פרטים ושלבים לתיקון השגיאה, ראה [שאין באפשרותך להיכנס ל-Microsoft Flow ומתקבלת שגיאת "אימות נכשל"](https://support.microsoft.com/help/4316891).
