# <a name="getting-an-invalid-license-error-in-outlook-on-the-web"></a>מקבל שגיאת רשיון לא חוקית ב-Outlook באינטרנט?

אם אתה משתמש ב-Outlook באינטרנט ואתה מקבל **משהו שגיאה שגויה** המכילה **X-OWA-שגיאה: Microsoft. Exchange. Data. InvalidLicenseException**, רשיון Exchange Online שלך אינו מוקצה כראוי או שפג תוקפו לאחרונה. מנהל המערכת יכול להקצות לך רשיון על-ידי ביצוע השלבים הבאים:
  
1. פתח את [מרכז הניהול של Microsoft 365](https://portal.office.com/adminportal/home#/homepage) ותחת **משתמשים פעילים**, בחר **ערוך משתמש**.

2. בדף **עריכת משתמש** שנפתח, בחר את המשתמש. בדף ' מאפייני משתמש ' שנפתח, לחץ על **ערוך** עבור **רשיונות מוצרים**.

3. בדף **רשיונות מוצר** שנפתח, בחר את ערך **המיקום** המתאים והקצה רשיון המכיל את Exchange Online (הרחב את הרשיון כדי לראות את הפרטים שלו). לסיום, לחץ על **שמור**.
