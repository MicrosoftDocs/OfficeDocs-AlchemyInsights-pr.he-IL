# <a name="learn-how-to-configure-agenda-settings"></a>למד כיצד להגדיר הגדרות סדר יום

הגדרות סדר יום ממוקמות בהגדרות **לוח השנה** ב-Outlook באינטרנט. לקבלת מידע נוסף, ראה [הגדרות לוח שנה ב-Outlook באינטרנט](https://support.office.com/article/12cba5a4-4f95-4d00-bfc3-b694aa67ac8f).

למנהלים יש אפשרות להשתמש ב **-Cmdlet Set-לוח** שנה ב-Exchange Online PowerShell כדי להגדיר הגדרות סדר יום. לקבלת מידע נוסף, ראה [הודעת הגדרת לוח](https://technet.microsoft.com/library/dd351284)שנה.
