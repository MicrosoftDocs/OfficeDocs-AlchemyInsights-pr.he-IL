# <a name="learn-how-to-configure-agenda-settings"></a>למד כיצד לקבוע תצורה של הגדרות סדר יום

הגדרות סדר יום ממוקמות בהגדרות **לוח השנה** ב-Outlook באינטרנט. לקבלת מידע נוסף, ראה [הגדרות לוח שנה ב-Outlook באינטרנט](https://support.office.com/article/12cba5a4-4f95-4d00-bfc3-b694aa67ac8f).

מנהלי מערכת יכולים להשתמש ב **-Cmdlet Set-CalendarNotification** ב-Exchange Online PowerShell כדי לקבוע את התצורה של הגדרות סדר יום. לקבלת מידע נוסף, ראה [Set-CalendarNotification](https://technet.microsoft.com/library/dd351284).
