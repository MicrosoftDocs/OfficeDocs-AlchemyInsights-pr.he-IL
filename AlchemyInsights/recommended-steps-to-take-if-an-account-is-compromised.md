[<span data-ttu-id="83fb7-103">וידאו: תיקון חשבון של Office 365 שנחשף לסכנה</span><span class="sxs-lookup"><span data-stu-id="83fb7-103">VIDEO: Fixing a compromised Office 365 account</span></span>](https://www.microsoft.com/videoplayer/embed/RE2jvOb?pid=ocpVideo0-innerdiv-oneplayer&amp;postJsllMsg=true&amp;maskLevel=20&amp;autoplay=true)
  
1. <span data-ttu-id="83fb7-104">[אפס את סיסמת המשתמש](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) באופן מיידי.</span><span class="sxs-lookup"><span data-stu-id="83fb7-104">[Reset the user's password](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) immediately.</span></span> <span data-ttu-id="83fb7-105">אל תמסור את הסיסמה החדשה למשתמש הקצה בדואר אלקטרוני.</span><span class="sxs-lookup"><span data-stu-id="83fb7-105">Do not communicate the new password through email to the end user.</span></span>

2. <span data-ttu-id="83fb7-106">הסר כל [כתובת להעברה](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) חשודה שהוגדרה ברמת תיבת הדואר.</span><span class="sxs-lookup"><span data-stu-id="83fb7-106">Remove any suspicious [forwarding addresses](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) set at the mailbox level.</span></span>

3. <span data-ttu-id="83fb7-107">הסר כל [כלל תיבת דואר נכנס](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) חשוד שהוגדר בתוך תיבת הדואר.</span><span class="sxs-lookup"><span data-stu-id="83fb7-107">Remove any suspicious [inbox rules](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) set within the mailbox.</span></span>

4. <span data-ttu-id="83fb7-108">אם האפשרות לשלוח דואר אלקטרוני חסומה עבור המשתמש, [עבור אל 'משתמשים מוגבלים' כדי לבטל את חסימת החשבון](https://protection.office.com/?hash=/restrictedusers).</span><span class="sxs-lookup"><span data-stu-id="83fb7-108">If the user is blocked from sending email, [go to the Restricted Users to unblock the account](https://protection.office.com/?hash=/restrictedusers).</span></span> <span data-ttu-id="83fb7-109">מרגע סיום התהליך, המשתמש יוכל לחזור ולשלוח הודעות תוך שעה.</span><span class="sxs-lookup"><span data-stu-id="83fb7-109">Once done, the user should be able to resume sending messages within 1 hour.</span></span>

5. <span data-ttu-id="83fb7-110">הסר את חשבון המשתמש מכל [קבוצת תפקידים ניהוליים](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504) עד שתהיה בטוח שהחשבון אינו חשוף עוד לסכנה.</span><span class="sxs-lookup"><span data-stu-id="83fb7-110">Remove the user account from any [administrative role groups](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504) until you are confident that the account is no longer compromised.</span></span>

<span data-ttu-id="83fb7-111">כדי למזער את הפוטנציאל של הפרת נתונים או חשיפת חשבון לסכנה בעתיד, מומלץ לקרוא את [מאמר שיטות העבודה המומלצות לאבטחה של Office 365](https://support.office.com/article/9295e396-e53d-49b9-ae9b-0b5828cdedc3).</span><span class="sxs-lookup"><span data-stu-id="83fb7-111">To minimize the potential of a data breach or a compromised account in the future, we recommend reading our [Office 365 Security best practices article](https://support.office.com/article/9295e396-e53d-49b9-ae9b-0b5828cdedc3).</span></span>
