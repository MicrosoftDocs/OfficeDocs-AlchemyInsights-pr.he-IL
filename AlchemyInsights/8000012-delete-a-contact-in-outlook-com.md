# <a name="delete-a-contact"></a>מחיקת איש קשר

1. בחלק העליון של הדף, בחר מפעיל יישום ![לחצן מפעיל היישום. ](media/9634bec0-78d1-4282-8aea-7c5e81f162d2.png) ולאחר מכן בחר **אנשים**. 
    
2. בחר את איש הקשר שברצונך למחוק ובחר **מחק**![למחוק](media/deb47846-8483-4f9d-813a-fc8fe288b583.png).
    
    אם אין באפשרותך לבחור **למחוק**, איש הקשר ייתכן Skype או חשבון רשת חברתית מחובר. כדי למחוק את איש הקשר, עבור Skype או לחשבון שבו ממוקם איש הקשר.
    
3. בחר ' **מחק** ' כדי לאשר.
