# <a name="what-is-microsoft-search-in-bing"></a>מהו חיפוש Microsoft ב-Bing?

[Microsoft Search ב-Bing](https://docs.microsoft.com/deployoffice/microsoft-search-bing#what-is-microsoft-search-in-bing) היא תכונה שעוזרת לך לחפש ברחבי הארגון ובאינטרנט. השתמש בשורת הכתובת ובתיבת החיפוש של Bing כדי למצוא מסמכים, אתרים פנימיים וחיצוניים, כלים עסקיים, אנשים ותרשימים ארגוניים, קבצים, אתרי SharePoint, תוכן OneDrive, Teams ו-קטרת שיחות ומשאבים רבים אחרים.
