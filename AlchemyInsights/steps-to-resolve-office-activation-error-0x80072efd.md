# <a name="steps-to-resolve-office-activation-error-0x80072efd"></a>שלבים לפתרון שגיאת ההפעלה של 0x80072EFD

נסה פתרונות אלה כדי לתקן את שגיאת ההפעלה של Office:
  
1. בדוק את [חומת האש](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckFirewall)שלך.

2. בדוק את [תוכנת האנטי-וירוס](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckAV)שלך.

3. בדיקת [הגדרות ה-proxy](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckProxy)

4. הורד והתקן את [מסייע התמיכה והשחזור](https://aka.ms/SARA-OfficeActivation-Alchemy) והפעלת **פותר בעיות ההפעלה**.
