# <a name="control-creation-of-microsoft-365-groups"></a>שליטה ביצירת קבוצות Microsoft 365

כברירת מחדל, כל המשתמשים בחברה יכולים ליצור קבוצות. במקרה הצורך, תוכל לשנות זאת ולהקצות את הזכות ליצור קבוצות לקבוצות של אנשים בחברה שלך. שים לב כי פעולה זו תשפיע על יצירת קבוצות מכל שירות של Microsoft 365 (כגון Outlook, Teams, Planner וכן הלאה).
  
הוראות: [ניהול מי יכול ליצור קבוצות Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
