# <a name="pst-data-files-in-outlookcom"></a>קבצי נתונים PST ב- Outlook.com

אין אפשרות לייבא של Outlook. קובץ נתונים PST לחשבונך Outlook.com באמצעות דפדפן.

כדי לייבא. קובץ PST הנתונים, עליך להשתמש app שולחן העבודה של Outlook. כדי ללמוד עוד, קרא [לייבא דואר אלקטרוני, אנשי קשר ולוח שנה מתוך ה-. pst ב- Outlook.](https://support.office.com/article/431a8e9a-f99f-4d5f-ae48-ded54b3440ac)
