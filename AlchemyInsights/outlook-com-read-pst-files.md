# <a name="pst-data-files-in-outlookcom"></a>PST קבצים נתונים ב Outlook.com

לא ניתן לייבא Outlook. PST קובץ הנתונים לחשבון Outlook.com שלך באמצעות דפדפן.

כדי לייבא. PST קובץ הנתונים, עליך להשתמש ביישום שולחן העבודה של Outlook. כדי ללמוד עוד, קרא את [ייבוא דואר אלקטרוני, אנשי קשר ולוח שנה מ-Outlook. pst.](https://support.office.com/article/431a8e9a-f99f-4d5f-ae48-ded54b3440ac?wt.mc_id=Office_Outlook_com_Alchemy)
