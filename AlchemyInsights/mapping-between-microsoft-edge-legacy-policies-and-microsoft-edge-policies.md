# <a name="learn-about--the-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>קבלת מידע אודות המיפוי בין מדיניות מדור קודם של Microsoft Edge ומדיניות של Microsoft Edge

מדיניות הקבוצה של Microsoft Edge Legacy (גירסאות 45 וגירסאות קודמות) והגדרות ניהול מכשירים ניידים (MDM) ממפים למדיניות הרלוונטית של Microsoft Edge, גירסה 80. לקבלת מידע נוסף, עיין [במיפוי מדיניות קצה של Microsoft edge למיקרוסופט](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge).
