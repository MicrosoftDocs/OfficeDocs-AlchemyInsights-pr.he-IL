# <a name="changing-shared-mailbox-send-as-behavior"></a>שינוי אופן הפעולה ' שלח כתיבת דואר משותפת '

כברירת מחדל, הודעות הנשלחות מתיבת הדואר המשותפת אינן נשמרות בתיקיה 'פריטים שנשלחו' של תיבת הדואר המשותפת. במקום זאת, הן נשמרות בתיקיה 'פריטים שנשלחו' של האדם ששלח את ההודעה.
  
אם ברצונך שכל החברים יראו את הדואר האלקטרוני שנשלח, במרכז הניהול של Microsoft 365, ערוך את פרטי תיבת הדואר המשותפת ובחר בפקודה \> **ערוך** **פריטים שנשלחו** .
  
כדי לקרוא עוד, ראה [יצירת תיבת דואר משותפת](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox).
