# <a name="project-web-app-is-missing-from-the-sharepoint-site-collection"></a>Project Web App חסר באוסף האתרים של SharePoint

ייתכן שבאוסף האתרים של SharePoint, Project Web App לא יהיה מהסיבות הבאות:

1. לארגונים יש רשיונות של ' יסודות מקוונים של פרוייקט ' בלבד. מנהלים יצטרכו להוסיף רישיון פרוייקט מקוון ו/או פרימיום. האתר יצא מקריאה בלבד בשלב זה. לקבלת מידע נוסף, ראה [השוואת פתרונות ניהול פרוייקטים](https://products.office.com/project/compare-microsoft-project-management-software?tab=1).
2. השוכר נתקל בבעיית הקצאה שבה הקצאת האתר של Project Web App נדחתה. אם חלפו יותר מ -24 שעות ואתה עדיין נתקל בבעיה, אנא רשום כרטיס תמיכה. אנחנו מבינים ש -24 שעות. נראה כמו זמן רב במקרים רבים, אנחנו כבר עובדים על פתרון. בבקשה תן לנו לפחות 24. שעות כדי להשלים פיתרון
