# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>פתרון בעיות בהגדרה של Outlook mobile עבור תיבת דואר משותפת

אין באפשרותך לגשת לתיבת הדואר המשותפת מתוך אפליקציית Outlook המותקנת במכשיר הנייד. האם אתה מעוניין בתכונה זו? אפשר לצוות ההנדסה של Outlook לדעת. [הצביעו כאן](https://go.microsoft.com/fwlink/?linked=862116)!
  
באפשרותך להשתמש בתיבת דואר משותפת במכשיר נייד בלבד על-ידי פתיחת הדפדפן, כניסה ולאחר מכן עבור אל Outlook באינטרנט. לקבלת הוראות, ראה [הוספת תיבת דואר משותפת ב-Outlook באינטרנט](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). ב- Outlook באינטרנט, תוכל לגשת לתיבת הדואר המשותפת.
  
מידע נוסף: [קבל עזרה באפליקציה עבור Outlook עבור iOS ו-Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
