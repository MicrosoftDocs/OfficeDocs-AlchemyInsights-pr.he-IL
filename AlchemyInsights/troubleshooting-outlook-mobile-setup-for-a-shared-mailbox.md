# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>פתרון בעיות בהתקנה הניידת של Outlook עבור תיבת דואר משותפת

אין באפשרותך לגשת לתיבת הדואר המשותפת מתוך אפליקציית Outlook המותקנת במכשיר הנייד. האם אתה מעוניין בתכונה זו? תן לצוות ההנדסה של Outlook לדעת. ! [הצביעו כאן](https://go.microsoft.com/fwlink/?linked=862116)
  
באפשרותך להשתמש רק בתיבת דואר משותפת במכשיר נייד על-ידי פתיחת הדפדפן, כניסה למחשב ולאחר מכן הולך ל-Outlook באינטרנט. לקבלת הוראות, ראה [הוספת תיבת דואר משותפת ב-Outlook באינטרנט](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). ב- Outlook באינטרנט, תוכל לגשת לתיבת הדואר המשותפת.
