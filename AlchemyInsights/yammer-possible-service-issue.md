# <a name="yammer---possible-service-issue"></a>קטרת-בעיית שירות אפשרית

ייתכן שקיימת בעיה זמנית שבה משתמשים עשויים להיתקל בשגיאות בעת גישה ל-קטרת. בדוק את [לוח המחוונים של תקינות השירות](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) כדי לראות אם פעולה זו משפיעה על הרשת שלך.
