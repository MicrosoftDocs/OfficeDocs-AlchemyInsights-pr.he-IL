# <a name="yammer---possible-service-issue"></a>Yammer - בעיית שירות אפשרית

ייתכן שיש בעיה זמנית שבה משתמשים עלולים להיתקל בשגיאות בעת גישה ל- Yammer. בדוק את [לוח המחוונים של תקינות](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) השירות כדי לראות אם הדבר משפיע על הרשת שלך.
