# <a name="permissions-and-public-folders"></a>הרשאות ותיקיות ציבוריות

באפשרותך לשנות את ההרשאות בתיקיות הציבוריות שלך באמצעות Outlook, מרכז הניהול של Exchange (EAC) או PowerShell:
  
- לקבלת הוראות של Outlook, [לחץ כאן](https://support.office.com/article/Set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005.aspx).
    
- עבור EAC, עיין [במאמר זה](https://technet.microsoft.com/library/jj651147%28v=exchg.150%29.aspx.aspx#Anchor_1) לקבלת הוראות. 
    
- עבור Powershell, עיין [במאמר זה](https://technet.microsoft.com/library/bb124743%28v=exchg.160%29.aspx.aspx) לקבלת הוראות לגבי השימוש ב-Add-PublicFolderClientPermission unifiedgroup. אם אתה זקוק להוראות כדי להתחבר ל-Exchange Powershell, לחץ [כאן](https://technet.microsoft.com/library/jj984289%28v=exchg.160%29.aspx.aspx).
    
אם **משתמשים חיצוניים אינם יכולים לשלוח הודעות דואר אלקטרוני לתיקיה ציבורית המותאמת לדואר**, הסיבה לכך עשויה להיות שהתיקיה הציבורית חסרה הרשאות הנדרשות למסירת דואר אלקטרוני חיצוני. באפשרותך לתקן זאת באמצעות ההוראות של Outlook [כאן](https://technet.microsoft.com/library/aa997560%28v=exchg.150%29.aspx.aspx#Anchor_1)או הוראות PowerShell [כאן](https://support.microsoft.com/help/2984402/-5.7.1-smtp-550-5.7.1-resolver.rst.authrequired-nondelivery-report-when-external-users-try-to-send-mail-to-mail-enabled-public-folders-in-office-365.aspx).
