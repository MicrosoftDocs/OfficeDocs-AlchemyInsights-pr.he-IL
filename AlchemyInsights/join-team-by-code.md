# <a name="join-a-team-by-code"></a>הצטרפות לצוות באמצעות קוד

Microsoft מודעת לבעיה שבה למשתמשים אין אפשרות ליצור קוד **הצטרפות לצוות** , או להצטרף לצוות באמצעות התכונה ' **הצטרף באמצעות קוד** '. כפתרון, בעלי צוות יכולים להוסיף משתמשים באופן ידני, לפי הצורך.
