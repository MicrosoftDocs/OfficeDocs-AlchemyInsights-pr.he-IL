# <a name="changes-to-external-user-access"></a>שינויים בגישת משתמש חיצונית

החל מ-23 במרץ 2018, משתמשים חיצוניים לא יוכלו עוד לגשת לתוכן ששותף עם כולם, כל המשתמשים המאומתים או כל הטפסים, גם אם הם הוזמנו או נוספו לקבוצות מורשות. משתמשים חיצוניים יוכלו רק לגשת לתוכן ששותף עם המשתמש או עם קבוצות שאליהן המשתמש שייך. אם ברצונך לשמור על ההגדרות הקיימות, נא ראה:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
