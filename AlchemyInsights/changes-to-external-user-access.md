# <a name="changes-to-external-user-access"></a>שינויים גישת משתמש חיצוני

החל 23 במרץ, 2018, משתמשים חיצוניים כבר לא תהיה אפשרות גישה לתוכן שותפה עם כולם, כל המשתמשים המורשים או כל המשתמשים טפסים, גם אם מוזמנים או להוסיף לקבוצות מורשים. משתמשים חיצוניים בלבד תהיה אפשרות גישה לתוכן שותפה עם המשתמש או עם הקבוצות אליהן שייך המשתמש. אם ברצונך לשמור את ההגדרות הקיימות, נא ראה:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
