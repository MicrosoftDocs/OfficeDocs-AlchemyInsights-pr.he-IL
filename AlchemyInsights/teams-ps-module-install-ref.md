# <a name="what-you-can-accomplish-with-microsoft-teams-powershell-module"></a>מה ניתן לבצע במודול Microsoft Teams PowerShell

אם עליך לדעת כיצד להתקין את המודול Teams PowerShell, באפשרותך לעיין [בבלוג זה](https://blogs.technet.microsoft.com/skypehybridguy/2017/11/07/microsoft-teams-powershell-support/).

לקבלת מידע נוסף:

- סקור את [הסקירה של teams PowerShell](https://docs.microsoft.com/MicrosoftTeams/teams-powershell-overview) כדי להבין את [יכולות תפקיד הניהול](https://docs.microsoft.com/MicrosoftTeams/using-admin-roles), [ליצור ולנהל צוותים באמצעות PowerShell](https://docs.microsoft.com/MicrosoftTeams/teams-powershell-overview#creating-and-managing-teams-via-powershell) [ולנהל מדיניות](https://docs.microsoft.com/MicrosoftTeams/teams-powershell-overview#managing-policies-via-powershell) או [תצורות](https://docs.microsoft.com/MicrosoftTeams/teams-powershell-overview#managing-configurations-via-powershell). 
- סקור את [מדריך הפקודות של Microsoft teams PowerShell](https://docs.microsoft.com/powershell/module/teams/?view=teams-ps) עבור רשימה של פקודות הזמינות באמצעות powershell.
