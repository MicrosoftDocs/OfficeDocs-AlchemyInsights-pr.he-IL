# <a name="learn-the-language-for-creating-advanced-hunting-queries"></a>למד את השפה ליצירת שאילתות ציד מתקדמות

חיפוש מתקדם מבוסס על [שפת השאילתה Kusto](https://go.microsoft.com/fwlink/?linkid=2144620). באפשרותך להשתמש בתחביר ובאופרטורים של Kusto כדי לבנות שאילתות שמאתרות מידע [בסכימה](https://go.microsoft.com/fwlink/?linkid=2144621) שנבנתה במיוחד לצורך חיפוש מתקדם.

לקבלת מידע נוסף, ראה [למד את שפת שאילתת הציד המתקדמת](https://go.microsoft.com/fwlink/?linkid=2144518).
