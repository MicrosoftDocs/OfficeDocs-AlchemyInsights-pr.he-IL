# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>המשולב במכשיר Windows 10 להגנת האיום המתקדמת של Microsoft Defender (MDATP)

MDATP תומך בכלי הפריסה ובשיטות הבאות עבור Windows 10:

- מדיניות קבוצתית
- Microsoft Endpoint Configuration Manager
- ניהול מכשירים ניידים, כולל Microsoft intune
- קבצי script מקומיים

לקבלת מידע נוסף, ראה [כלים ושיטות המשולבים עבור מכשירי Windows 10](https://go.microsoft.com/fwlink/?linkid=2143460).
