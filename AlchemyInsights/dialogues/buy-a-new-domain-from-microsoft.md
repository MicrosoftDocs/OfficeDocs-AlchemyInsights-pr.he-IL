# <a name="buy-a-new-domain-from-microsoft"></a>קניית תחום חדש מ-Microsoft

כדי לקנות תחום חדש מ-Microsoft, בצע את השלבים הבאים:

1. נווט אל **הגדרת תחומים >** ולחץ על **קנה תחום**. 
2. בצע את ההוראות באשף.
