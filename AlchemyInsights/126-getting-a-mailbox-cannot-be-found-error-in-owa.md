# <a name="getting-a-mailbox-not-found-error-in-outlook-on-the-web"></a><span data-ttu-id="ee5e8-102">קבלת דואר לא נמצאה שגיאה ב- Outlook באינטרנט?</span><span class="sxs-lookup"><span data-stu-id="ee5e8-102">Getting a mailbox not found error in Outlook on the web?</span></span>

<span data-ttu-id="ee5e8-p101">אם אתה משתמש ב- Outlook באינטרנט ולקבל הודעת שגיאה **לא היתה אפשרות למצוא את תיבת הדואר עבור** , לחשבון שבו השתמשת כדי להתחבר ל- Outlook באינטרנט אין רשיון של Exchange Online, ולכן, תיבת הדואר לא משויך לתיק הלקוח. המנהל שלך באפשרותך להקצות רשיון לחשבון שלך על-ידי ביצוע השלבים הבאים:</span><span class="sxs-lookup"><span data-stu-id="ee5e8-p101">If you're using Outlook on the web and you get a **Mailbox couldn't be found for** error, the account that you used to connect to Outlook on the web doesn't have an Exchange Online license and therefore, no mailbox is associated with the account. Your admin can assign a license to your account by following these steps:</span></span> 
  
1. <span data-ttu-id="ee5e8-105">פתח את [מרכז הניהול של Office 365](https://portal.office.com/adminportal/home#/homepage) ובחר תחת **משתמשים פעילים**, **ערוך משתמש**.</span><span class="sxs-lookup"><span data-stu-id="ee5e8-105">Open the [Office 365 admin center](https://portal.office.com/adminportal/home#/homepage) and under **Active users**, select **Edit a user**.</span></span>
    
2. <span data-ttu-id="ee5e8-p102">בדף **עריכת משתמש** שנפתחת, בחר את המשתמש. בעמוד מאפייני המשתמש שנפתחת, לחץ על **ערוך** עבור **רשיונות למוצר**.</span><span class="sxs-lookup"><span data-stu-id="ee5e8-p102">In the **Edit a user** page that opens, select the user. In the user properties page that opens, click **Edit** for **Product licenses**.</span></span>
    
3. <span data-ttu-id="ee5e8-p103">בדף **רשיונות מוצר** שנפתחת, בחר את הערך המתאים של **מיקום** ולהקצות רשיון המכיל Exchange Online (הרחב את הרשיון כדי לראות את הפרטים שלו). לאחר שסיימת, לחץ על **שמור**.</span><span class="sxs-lookup"><span data-stu-id="ee5e8-p103">In the **Product licenses** page that opens, select the appropriate **Location** value, and assign a license that contains Exchange Online (expand the license to see its details). When you're finished, click **Save**.</span></span>
