# <a name="learn-about-site-compatibility-affecting-changes-coming-to-microsoft-edge"></a>למד אודות תאימות אתר-המשפיעים על שינויים המגיעים ל-Microsoft Edge

האינטרנט מתפתח באופן קבוע כדי לשפר את חוויית המשתמש, את האבטחה והפרטיות. כמה שינויים משמעותיים מספיק כדי להשפיע על הפונקציונליות של עמודים קיימים. צוות הקצה של Microsoft מעדכן את העמודים כאשר החשיבה מתפתחת, הזמנים מתקשים [והשינויים הוכרזו](https://go.microsoft.com/fwlink/?linkid=2135534).
