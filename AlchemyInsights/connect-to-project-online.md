# <a name="connect-to-project-online"></a>התחבר ל-Project Online

באפשרותך להתחבר ל-Project Online באמצעות:

- לקוח ' שולחן עבודה של project Online ' כדי להתחבר ל-Project Online בסביבת Office 365. לקבלת מידע נוסף, ראה [התחברות ל-Project online עם לקוח שולחן העבודה של Project online](https://docs.microsoft.com/projectonline/connect-to-project-online-with-the-project-online-desktop-client).  
- שירות ה-BI Power כדי לבטל את נעילת התובנה מ-Project Web App כדי לסייע בניהול פרוייקטים, תיקי ומשאבים. לקבלת מידע נוסף, ראה [התחברות ל-Project Web App עם הכוח BI](https://docs.microsoft.com/power-bi/service-connect-to-project-online).
