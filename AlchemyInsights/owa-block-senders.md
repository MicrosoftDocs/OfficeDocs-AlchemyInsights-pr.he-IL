# <a name="block-or-unblock-senders"></a>חסימה או ביטול חסימה של שולחים

**מתיבת הדואר הנכנס שלך:**

- ברשימת ההודעות, בחר את ההודעה ולאחר מכן בחר **זבל** > **בלוק**.

**מתוך הגדרות:**

1. בחלק העליון של הדף, בחר בסמל גלגלי שיניים כדי לפתוח את הגדרות ולאחר מכן בחר **להציג את כל ההגדרות של Outlook**.
2. בחר **דואר** > **דואר "זבל" אלקטרוני**.
3. תחת **שולחים חסומים ותחומים**, בחר באפשרות ' **הוסף**'.
4. הקלד את כתובת הדואר האלקטרוני של התחום שברצונך לחסום ולאחר מכן הקש Enter.
5. בחר **שמור**.

כדי לבטל את החסימה של השולח, בחר בסמל פח לצד כתובת דואר אלקטרוני או קבוצת המחשבים שברצונך לבטל את החסימה.

לקבלת מידע נוסף, ראה [חסימת שולחים או ביטול חסימה של שולחים ב- Outlook באינטרנט](https://support.office.com/article/9bf812d4-6995-4d19-901a-76d6e26939b0).
