# <a name="change-your-profile-name"></a>שנה את שם הפרופיל שלך

1. עבור אל [פרטי חשבון Microsoft שלך](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. בחר **שם עריכה**. 
    
3. הזן שם פרטי ושם משפחה ובחר באפשרות ' **שמור**'. 
    
נא המתן עד 24 שעות לשינוי שם הפרופיל שלך כדי שייכנסו לתוקף.
  
לקבלת מידע נוסף, ראה [שינוי התמונה או השם של הפרופיל שלך ב-Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110).
