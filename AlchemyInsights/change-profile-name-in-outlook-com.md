# <a name="change-your-profile-name"></a>שינוי שם הפרופיל שלך

1. עבור אל [פרטי חשבון Microsoft שלך](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. בחר **ערוך שם**. 
    
3. הזן שם פרטי ושם משפחה, ובחר **שמור**. 
    
אפשר עד 24 שעות לשינוי שם הפרופיל שלך כדי להיכנס לתוקף.
  
לקבלת מידע נוסף, ראה [שינוי תמונת הפרופיל או שם הפרופיל שלך ב-Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110).
