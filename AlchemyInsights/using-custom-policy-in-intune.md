# <a name="using-custom-policy-in-intune"></a>באמצעות מדיניות מותאמת אישית ב- Intune

Intune יש מוכללות רבות של הגדרות עבור התקנים Android, iOS, macOS ו- Windows. באפשרותך גם ליצור פרופיל מותאם אישית ולאחר מכן לפרוס הגדרות מותאמות אישית אלו להתקני שלך באמצעות Intune. פרופילים מותאמים אישית נועדו להוסיף הגדרות התקן ותכונות אינם מוכללים Intune.

[הגדרות מותאמות אישית עבור התקנים Android ב- Intune](https://docs.microsoft.com/intune/custom-settings-android)

[הגדרות מותאמות אישית עבור ארגון Android התקנים Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[הגדרות מותאמות אישית עבור התקנים iOS ב- Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[הגדרות מותאמות אישית עבור התקנים macOS ב- Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[הגדרות מותאמות אישית עבור התקנים Windows 10 Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[הגדרות מותאמות אישית עבור Windows ההולוגרפית להתקנים ביקור ב- Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
