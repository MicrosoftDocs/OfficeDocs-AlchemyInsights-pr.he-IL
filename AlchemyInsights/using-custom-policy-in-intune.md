# <a name="using-custom-policy-in-intune"></a>שימוש במדיניות מותאמת אישית ב-Intune

Intune יש הגדרות רבות מובנית עבור אנדרואיד, iOS, macOS ו-Windows התקנים. באפשרותך גם ליצור פרופיל מותאם אישית ולאחר מכן לפרוס הגדרות מותאמות אישית אלה בהתקנים שלך באמצעות Intune. פרופילים מותאמים אישית מיועדים להוסיף הגדרות ותכונות של התקן שאינן מובנות ל-Intune.

[הגדרות מותאמות אישית עבור התקני אנדרואיד ב Intune](https://docs.microsoft.com/intune/custom-settings-android)

[הגדרות מותאמות אישית עבור אנדרואיד התקנים ארגוניים ב Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[הגדרות מותאמות אישית עבור מכשירי iOS ב Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[הגדרות מותאמות אישית עבור התקני macOS ב Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[הגדרות מותאמות אישית עבור התקני Windows 10 ב-Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[הגדרות מותאמות אישית של Windows הולוגרפית עבור התקנים עסקיים ב Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
