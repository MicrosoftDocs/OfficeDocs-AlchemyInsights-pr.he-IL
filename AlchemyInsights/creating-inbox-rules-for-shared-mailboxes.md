# <a name="creating-inbox-rules-for-shared-mailboxes"></a>יצירת כללי תיבת דואר נכנס עבור תיבות דואר משותפות

באפשרותך להוסיף כללי תיבת דואר משותפת יש לך גישה באופן דומה כיצד להוסיף כללי לחשבון משלהם. 
  
ודא כי אתה חבר של תיבת הדואר המשותפת, ולאחר מכן בצע את הפעולות המתוארות במאמר זה: [הוסף כללים לתיבת דואר משותפת](https://support.office.com/article/b0963400-2a51-4c64-afc7-b816d737d164 )
