# <a name="viewing-chat-history-in-microsoft-teams"></a>הצגת היסטוריית צ'אט ב- Microsoft Teams

כדי להציג את היסטוריית [](https://sip.protection.office.com/contentsearchbeta?ContentOnly=1) הצ'אטים של Teams, באפשרותך להשתמש [בחיפוש תוכן ממרכז](https://sip.protection.office.com/insightdashboard)האבטחה & תאימות , שבה תוכל לחפש בתיבת הדואר המשויכת לקבוצה של Microsoft 365 או לצוות Microsoft. 

ראה [חיפוש בקבוצות Microsoft Teams ו- Microsoft 365](https://docs.microsoft.com/microsoft-365/compliance/content-search) כדי ללמוד על יכולות החיפוש ממרכז האבטחה והתאימות. 

**הערה:** גבה, ייצוא, הדפסה ואחסון בארכיון של היסטוריית שיחות אינו זמין כעת ישירות מ- Microsoft Teams. הצבע על יכולת זו ב- [UserVoice.](https://microsoftteams.uservoice.com/forums/555103-public/suggestions/16982542-backup-export-printing-archive-options?page=2&per_page=20)
