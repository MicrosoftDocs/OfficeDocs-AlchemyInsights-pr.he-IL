# <a name="solutions-for-unsupported-operating-system"></a>פתרונות עבור "מערכת הפעלה לא נתמכת"

כאשר אתה מנסה להתקין את Microsoft 365, ייתכן שתראה הודעת שגיאה של **מערכת הפעלה לא נתמכת** אם המכשיר שלך אינו עומד [בדרישות המערכת](https://products.office.com/office-system-requirements).
  
Microsoft 365 דורש:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows server 2016, Windows Server 2012 R2, Windows Server 2012 או Windows Server 2008 R2

    או

- Mac OS X 10.10

אם קיבלת הודעה זו בעת ניסיון להתקין את Office במכשיר נייד או ב-Chromebook, ראה [כיצד להתקין ולנהל את Microsoft Office ב-Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
