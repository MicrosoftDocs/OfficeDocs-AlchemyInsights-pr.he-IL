# <a name="solutions-for-unsupported-operating-system"></a>פתרונות עבור "מערכת הפעלה שאינה נתמכת"

כאשר אתה מנסה להתקין את Office 365, ייתכן שתראה הודעת שגיאה של **מערכת הפעלה שאינה נתמכת** אם ההתקן שלך אינו עומד [בדרישות המערכת](https://products.office.com/office-system-requirements).
  
Office 365 דורש:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, שרת Windows 2016, Windows Server 2012 R2, שרת Windows 2012 או Windows Server 2008 R2

    או

- Mac OS X 10.10

אם התקבלה ההודעה הבאה בעת ניסיון להתקין את Office מכשיר נייד או של Chromebook, ראה [כיצד להתקין ולהפעיל את Microsoft Office על Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
