# <a name="live-events-in-microsoft-stream"></a>אירועים חיים ב-Microsoft Stream

באפשרותך ליצור אירועים חיים באמצעות Microsoft Stream ברחבי הארגון. באפשרותך לתזמן, להפיק ולספק אירועים חיים עבור מגוון תרחישים כגון אירועים ברחבי החברה, עדכוני מנהיגות ועוד.

- [יצירת אירוע חי ב-Microsoft Stream](https://docs.microsoft.com/stream/live-create-event)
- [תחילת התחלת האירועים החיים בצוותים של Microsoft](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [קביעת תצורה של אנקודרים עבור הזרמה חיה בזרם של Microsoft](https://docs.microsoft.com/stream/live-encoder-setup)
