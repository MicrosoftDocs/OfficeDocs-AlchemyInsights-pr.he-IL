# <a name="need-to-create-an-email-signature"></a>צריך ליצור חתימת דואר אלקטרוני?

אם יש לך חשבון Outlook או של Outlook את חשבון האינטרנט (Office 365 בעבודה או בבית הספר), באפשרותך ליצור חתימה ו להוסיף אותו באופן אוטומטי לכל ההודעות היוצאות, או באופן סלקטיבי להוסיף אותה להודעות יוצאות ספציפיים. שים לב כי חתימת דואר אלקטרוני שלך שונה ב- Outlook וב -Outlook באינטרנט. כדי ליצור את חתימת הדואר האלקטרוני שלך, עיין [צור והוסף חתימה על הודעות ב- Outlook](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) או [צור והוסף חתימת דואר אלקטרוני ב- Outlook באינטרנט](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx).
