# <a name="need-to-create-an-email-signature"></a>צריך ליצור חתימת דוא ל?

אם ברשותך חשבון Outlook או בחשבון Outlook באינטרנט (עבודה או בית ספר), באפשרותך ליצור חתימה ולהוסיפו באופן אוטומטי לכל ההודעות היוצאות, או להוסיפו בררנית להודעות יוצאות ספציפיות. שים לב שחתימת הדואר האלקטרוני שלך שונה ב-Outlook וב-Outlook באינטרנט. כדי ליצור את חתימת הדואר האלקטרוני שלך, ראה [יצירה והוספה של חתימה להודעות ב-outlook](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) או [יצירה והוספה של חתימת דוא ל ב-outlook באינטרנט](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx).
