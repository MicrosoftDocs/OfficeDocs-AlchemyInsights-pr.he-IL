# <a name="clear-the-browsing-data-stored-in-the-cloud-if-youve-turned-on-sync-or-personalization"></a>נקה את נתוני הגלישה המאוחסנים בענן (אם הפעלת סינכרון או התאמה אישית)

כדי להציג, לנהל או למחוק את הנתונים ששמרת בענן, השתמש בלוח המחוונים של הפרטיות. הנתונים שאתה מוחק באמצעות לוח המחוונים לא יימחקו מהמכשיר שלך. לקבלת פרטים, ראה [הצגת הנתונים בלוח המחוונים של הפרטיות.](https://support.microsoft.com/windows/view-your-data-on-the-privacy-dashboard-03d3e27f-1981-5ff4-ba1c-d6b1031ae433)
