# <a name="download-and-share-updates"></a>הורד ושתף עדכונים

במקום להוריד עדכונים בנפרד, מצב ההורדה של מיטוב המסירה מאפשר שיתוף בין מחשבים מרובים. לקבלת מידע נוסף, ראה [מיטוב מסירה עבור עדכונים של Windows 10](https://docs.microsoft.com/windows/deployment/update/waas-delivery-optimization).  

מצבי הורדה זמינים כוללים:  
- HTTP בלבד  
- LAN  
- קבוצה  
- אינטרנט  
- Simple  
- עקוף
