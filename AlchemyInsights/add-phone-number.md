# <a name="add-phone-number"></a>הוספת מספר טלפון

לפני שתוכל להגדיר משתמשים בארגון שלך שיקבלו ויבצעו שיחות, עליך [לקבל מס טלפון](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview) עבורם. סיפקנו לך תהליך שלב אחר שלב להשלמת ההגדרה של מערכת הטלפון. סקור את [הגדרת מערכת טלפון בארגון שלך](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview).
