# <a name="change-your-account-picture"></a>שינוי תמונת החשבון שלך

1. מעבר אל [מידע חשבון Microsoft שלך](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. בחר **שינוי תמונה**. 
    
3. בחר **תמונה חדשה**. 
    
4. בחר את התמונה שבה ברצונך להשתמש ובחר **פתוח**. 
    
5. גרור או שינוי הגודל של תיבת תמונה ובחר **שמור**. 
    
לתשומת לבך, עד 24 שעות כדי שהשינויים שלך תמונות הפרופיל או שם תיכנס לתוקף.
