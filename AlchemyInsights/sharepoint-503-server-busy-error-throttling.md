# <a name="sharepoint-online-throttling"></a>ויסות מקוון ב- SharePoint

משתמשים עלולה להופיע 503 של השרת הוא עסוק שגיאה בעת ניסיון לנווט לאתרי SharePoint או OneDrive. 

שגיאה זו יכול להיגרם על-ידי ויסות בתוך השירות של SharePoint. SharePoint Online משתמשת ויסות כדי לשמור על ביצועים מיטביים ואמינות של השירות המקוון של SharePoint. ויסות מגבלות מספר פעולות המשתמש או בו-זמניים שיחות (על-ידי script או קוד) כדי למנוע שימוש יתר של משאבים. אם אתה מקבל בוויסות, 99% מהזמן הוא עקב קוד מותאם אישית.

לקבלת מידע נוסף אודות ויסות [המנעות תחילת בוויסות או חסומים ב- SharePoint Online](https://docs.microsoft.com/sharepoint/dev/general-development/how-to-avoid-getting-throttled-or-blocked-in-sharepoint-online), ראה.

אם אתה סבור שגיאה זו אינו קשור אליהם ויסות, באפשרותך לבדוק אם קיימת פעילות תחזוקה המתרחשים דיירים שלך על-ידי ניווט אל [מרכז הודעה](https://portal.office.com/adminportal/home#/MessageCenter).

 לבסוף, ודא שבקר בדף [תקינות שירות](https://portal.office.com/adminportal/home#/servicehealth) לבדיקת עלוני יידוע/באירועים אשר ייתכן המתרחשים.
