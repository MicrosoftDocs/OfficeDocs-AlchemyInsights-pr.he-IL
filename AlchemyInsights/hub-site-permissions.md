# <a name="hub-site-permissions"></a>הרשאות אתר Hub

ניהול ההרשאות של אתר רכזת תלוי בסוג האתר המשמש כבסיס. אם האתר הוא אתר צוות המחובר לקבוצה, עליך לנהל הרשאות דרך הקבוצה Microsoft 365. אם זהו אתר תקשורת, עליך לנהל הרשאות באמצעות קבוצות SharePoint.

- [הרשאות אתר Hub](https://docs.microsoft.com/sharepoint/modern-experience-sharing-permissions#hub-site-permissions)  
- [תכנון אתרי Hub](https://docs.microsoft.com/sharepoint/planning-hub-sites)
