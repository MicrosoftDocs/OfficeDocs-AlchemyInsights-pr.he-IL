# <a name="open-and-use-a-shared-mailbox-in-outlook-on-the-web"></a>פתיחת תיבת דואר משותפת ושימוש בה ב- Outlook באינטרנט

תיבות דואר משותפות שאין אליהן גישה אינן מוצגות באופן אוטומטי ב- Outlook באינטרנט (נקרא בעבר Outlook Web App).

ניתן להציג את תיבת הדואר המשותפת ואת תיבת הדואר הראשית שלך ביחד או לפתוח את תיבת הדואר המשותפת בחלון דפדפן נפרד. כדי לקבל הודעות חדשות עבור תיבת הדואר המשותפת, עליך לפתוח את תיבת הדואר המשותפת בחלון דפדפן נפרד. לקבלת מידע נוסף, ראה [פתיחת תיבת דואר משותפת ושימוש בה ב- Outlook באינטרנט](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207).
