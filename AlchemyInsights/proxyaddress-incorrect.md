# <a name="proxyaddress-incorrect"></a>ProxyAddress שגויה

כאשר אובייקט מסונכרן כדי AD תכלת הרקיע, הערכים שמצוינים בתכונה proxyAddresses ב- Active Directory מושווים עם כללים AD תכלת הרקיע ולאחר מכן התכונה proxyAddresses מאוכלס ב- AD תכלת הרקיע. לכן, ערכי התכונה proxyAddresses עבור האובייקט ב- Active Directory עשויים שלא להיות זהים לערכים של תכונת proxyAddresses ב- AD תכלת הרקיע.
  
כדי ללמוד עוד אודות כיצד לאכלס את proxyaddress, ראה [כיצד התכונה proxyAddress מאוכלס ב- AD תכלת הרקיע](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
