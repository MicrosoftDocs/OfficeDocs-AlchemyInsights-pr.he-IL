# <a name="proxyaddress-incorrect"></a>כתובת ProxyAddress גויה

כאשר אובייקט מסונכרן לתכלת AD, הערכים המצוינים בתכונה proxyAddresses ב-Active Directory מושווים עם כללי כחול-לספירה ולאחר מכן התכונה proxyAddresses תובות מאוכלסת בתכלת AD. משום כך, ייתכן שהערכים של התכונה proxyAddresses עבור האובייקט ב-Active Directory לא יהיו זהים לערכים של התכונה proxyAddresses בתכלת המודעה.
  
כדי ללמוד עוד אודות אופן האכלוס של כתובת ה-proxyaddress ראה [כיצד התכונה proxyAddress מאוכלסת בתכלת המודעה](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
