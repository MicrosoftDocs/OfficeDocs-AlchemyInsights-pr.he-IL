# <a name="block-ads-in-the-microsoft-edge-browser-on-an-android-or-ios-device"></a>חסימת מודעות בדפדפן Microsoft Edge במכשיר Android או iOS

כדי לחסום את כל המודעות למעט אלה הנחשבות למקובלות מאת AdBlock Plus:
1. בחר **.** .. > **(הגדרות ועוד)**  >  **הגדרות**  >  **חוסמי תוכן**.
2. הפעלת **מודעות לחסימות**.

כדי לחסום את כל המודעות:
1. בחר **.** .. > **(הגדרות ועוד)**  >  **הגדרות**  >  **חוסמי תוכן**.
2. הפעלת **מודעות לחסימות**.
3. בחר **הגדרות מתקדמות**.
4. בטל את **האפשרות אפשר מודעות מקובלות**.
