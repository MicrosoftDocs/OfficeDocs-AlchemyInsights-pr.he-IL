# <a name="send-email-from-an-alias-or-secondary-address"></a>שליחת דואר אלקטרוני מכינוי או מכתובת משנית

Microsoft 365 אינו תומך כעת בשליחת דואר אלקטרוני מכינוי או ככתובת דואר אלקטרוני משנית. ניתן להשתמש בכינויי דואר אלקטרוני כדי לקבל הודעות דואר אלקטרוני בלבד. אם ברצונך לשלוח דואר אלקטרוני מכתובת דואר אלקטרוני אחרת, עליך ליצור תיבת דואר משותפת. תיבת הדואר המשותפת תספק לך כתובת דואר אלקטרוני אחרת, מבלי לדרוש רשיון נוסף.
  
- [מעבר לכאן כדי ליצור תיבת דואר משותפת](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)

באפשרותך גם לקרוא את המאמר [יצירת תיבת דואר משותפת](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox) שתספר לך כל מה שעליך לדעת אודות תיבות דואר משותפות, כולל אופן השימוש בהן.
