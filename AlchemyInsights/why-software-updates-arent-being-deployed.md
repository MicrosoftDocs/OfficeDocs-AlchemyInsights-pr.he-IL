# <a name="why-software-updates-are-not-being-deployed"></a>מדוע עדכוני תוכנה אינם נפרסים?

סקור את הפרטים הבאים אם קבעת את התצורה של עדכוני תוכנה באמצעות הטבעת עדכון של Windows 10, אך העדכונים אינם נפרסים:  

- שקול לשנות את שירות השירות של Windows מסוג ההפצה  **החצי שנתי**  לסוג הפצה מחמיר ותדיר יותר  
- בדוק את התקופה הדחייה עבור עדכון  **איכות**  **ועדכון תכונות**. משך הזמן הנדחה עשוי לגרום לעיכוב בעדכונים עבור 180 ימים.
