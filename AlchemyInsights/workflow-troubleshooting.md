# <a name="workflow-troubleshooting"></a>פתרון בעיות של זרימת עבודה

לקבלת מידע אודות איתור באגים של זרימות עבודה של SharePoint, סקור את הפעולות הבאות:
- איתור באגים של זרימות עבודה דומה לזו של גירסאות קודמות.  לקבלת מידע נוסף, ראה [איתור באגים של SharePoint זרימות עבודה](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows).
- לקבלת מידע נוסף אודות פתרון בעיות, ראה [פתרון בעיות של SharePoint Server שגיאות אימות של זרימת עבודה](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio).
