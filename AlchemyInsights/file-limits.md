# <a name="file-upload-limits-in-sharepoint-online"></a>מגבלות העלאה של קבצים ב-SharePoint Online

- ספריה יכולה לכלול עד 30,000,000 קבצים ותיקיות.
    
- האורך הכולל של כתובת ה-URL, כולל שם הקובץ, אינו יכול לחרוג מ-400 תווים.
    
- כל קובץ מצומצם מ-15 GB. קבצים המצורפים לפריטי רשימה עשויים להיות בגודל של עד 250 MB.
    
- לקבלת סינכרון ביצועים מיטבי באמצעות לקוח הסינכרון החדש של OneDrive, מומלץ לאחסן רק קבצי 100,000 בספריה.
