# <a name="limit-sharing-in-microsoft-365"></a>הגבל שיתוף ב-Microsoft 365

בעוד שאין באפשרותך להשבית לחלוטין שיתוף פנימי, או להסיר את הלחצן שתף מאתרים, קיימות מגוון דרכים בהן באפשרותך להגביל את השיתוף ב-Microsoft 365 כדי לענות על הצרכים של הארגון שלך. לקבלת מידע נוסף, ראה [הגבלת שיתוף ב-Microsoft 365](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing).
