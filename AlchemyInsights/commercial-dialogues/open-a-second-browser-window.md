# <a name="open-a-second-browser-window"></a>פתיחת חלון דפדפן שני

כדי לפתוח חלון דפדפן שני, בצע את השלבים הבאים:

1. בחלון הדפדפן הראשון (שבו אתה קורא את זה), ודא שאשף התחומים פתוח. אם לא לחצת על הלחצן לעיל כדי להפעיל את אשף התחומים, באפשרותך להגיע לאשף התחומים על-ידי מעבר אל **תחומים > הוספת תחום**.

2. פתיחת חלון דפדפן שני. בדפדפן השני, תוכל לבצע את השלבים הבאים.
