# <a name="problem-verifying-a-domain"></a>בעיה באימות תחום

אם Office 365 אינו מאמת את התחום שלך בשעה או שתיים, ישנה בעיה.

בחר **לא** בהנחיות הבאות כדי לציין שלא **פתרתי את הבעיה** ולאחר מכן תראה את האפשרויות לשיחה עם אדם.
