# <a name="refresh-using-web-connector-doesnt-work-properly"></a>רענון באמצעות מחבר האינטרנט אינו פועל כהלכה

אם יש לך קובץ script של מחבר אינטרנט המשתמש בפונקציה [web. Page](https://msdn.microsoft.com/library/mt260924.aspx) , ועדכנת את ערכת הנתונים או את הדוח שלך לאחר ה-18 בנובמבר, 2016, עליך להשתמש בשער כדי שרענון יפעל כהלכה.

לקבלת מידע נוסף: [https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
