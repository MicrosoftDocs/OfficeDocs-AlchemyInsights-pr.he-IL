# <a name="how-to-type-special-characters-on-a-mac"></a>כיצד להקליד תווים מיוחדים ב-Mac

כדי להקליד תווים מיוחדים במחשב ה-Mac שלך, פתח את תפריט ' **תווים מיוחדים** ' באמצעות קיצור המקשים ' ⌘ בקרת **הפקודה '**  +  **Control**  +  **Space**.

קיצור מהיר עבור @ גם הוא **ALT**  +  **64**.
