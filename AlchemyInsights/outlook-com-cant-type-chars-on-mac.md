# <a name="how-to-type-special-characters-on-a-mac"></a>כיצד להקליד תווים מיוחדים ב- Mac

כדי להקליד תווים מיוחדים ב- Mac שלך, פתח את תפריט **תווים מיוחדים** עם קיצור המקשים ⌘ **הפקודה** + **שליטה** + **שטח**.

קיצור דרך מהירה עבור @ הוא גם **ALT** + **64**.
