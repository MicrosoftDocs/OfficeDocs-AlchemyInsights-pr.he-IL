# <a name="how-to-select-all-in-outlookcom"></a>כיצד "בחר הכל" ב- Outlook.com

באפשרותך לבחור רק כל אם כל ההודעות מוצגות. הדבר נועד למנוע פעולות בשוגג על הודעות שאתה רואה.

**CTRL + לחיצה על** בוחר מספר הודעות בודדות.

**SHIFT + לחיצה** בוחרת את כל ההודעות בין ההודעה הראשונה והאחרונה שתבחר.
