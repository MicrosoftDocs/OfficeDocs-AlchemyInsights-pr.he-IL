# <a name="how-to-select-all-in-outlookcom"></a>כיצד לבחור באפשרות ' בחר הכל ' ב-Outlook.com

באפשרותך לבחור הכל רק אם כל ההודעות גלויות. זאת כדי למנוע פעולות בטעות בהודעות שאינך יכול לראות.

**CTRL + לחץ על** בחירת הודעות בודדות מרובות.

**SHIFT + לחיצה על** בחירת כל ההודעות בין ההודעה הראשונה וההודעה האחרונה שבחרת.
