# <a name="microsoft-teams---guest-access"></a>צוותי מיקרוסופט-גישה לאורחים

אם אתה זקוק לעזרה בתקשורת עם משתמשים מחוץ לארגון שלך בצוותים, עליך להחליט אם להשתמש [בגישה לאורחים או בגישה חיצונית (פדרציה)](https://docs.microsoft.com/en-us/microsoftteams/manage-external-access#external-access-vs-guest-access), או כדי להפוך את שניהם לזמינים.

קיימים מספר הבדלים חשובים בין תכונות אלה. ייתכן שתעדיף את הגישה לאורחים בגלל ערכת תכונות רחבה יותר. עם זאת, גישה חיצונית (פדרציה) היא מצויינת לתקשורת פשוטה יותר עם משתמשים חיצוניים. הקפד [לסקור את ההבדלים](https://docs.microsoft.com/en-us/microsoftteams/manage-external-access#external-access-vs-guest-access) ולבחור את האחד המתאים ביותר לצרכים העסקיים שלך.

אם ברצונך לאפשר גישה לאורחים, ודא שעקבת אחר כל השלבים [ברשימת הפעולות לביצוע של גישה לאורחים](https://docs.microsoft.com/en-us/microsoftteams/guest-access-checklist). לאחר שתסיים לגמרי, יהיה עליך [להמתין עד 24 שעות](https://docs.microsoft.com/en-us/microsoftteams/manage-guests#guest-access-latencies) כדי שההגדרות ייכנסו לתוקף.

אם אתה בטוח שהשלמת את כל השלבים ברשימת הפעולות לביצוע, ועברו יותר מ -24 שעות, המשך לנסות [להוסיף אורח לצוות שלך](https://support.office.com/en-us/article/add-guests-to-a-team-in-teams-fccb4fa6-f864-4508-bdde-256e7384a14f#ID0EAABAAA=Desktop).

אם ברצונך גם לאפשר גישה חיצונית (פדרציה), בצע את כל השלבים ב- [אפשר למשתמשי הצוותים לשוחח ולתקשר עם משתמשים בארגון אחר](https://docs.microsoft.com/en-us/microsoftteams/manage-external-access#let-your-teams-users-chat-and-communicate-with-users-in-another-organization).

לקבלת מידע נוסף, כולל סרטוני כיצד [לבצע, ראה גישה לאורחים בצוותי Microsoft](https://docs.microsoft.com/microsoftteams/guest-access).
