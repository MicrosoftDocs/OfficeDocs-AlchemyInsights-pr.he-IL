# <a name="get-support-with-microsoft-365"></a>קבל תמיכה ב- Microsoft 365

כדי למצוא את המשאבים המתאימים לבעיה שלך, נצטרך לקבל מידע מפורט יותר. ציין שוב את הבעיה וכלול שמות מוצרים, תכונות או שגיאות ספציפיים ושלח שוב את השאילתה.
