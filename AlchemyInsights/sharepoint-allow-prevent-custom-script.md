# <a name="allow-custom-script-to-enable-features"></a>אפשר בקובץ script מותאם אישית להפוך תכונות לזמינות

מנהלים ייתכן שיהיה עליך **לאפשר קובץ Script מותאם אישית** כדי לאפשר תכונות מסוימות. לדוגמה, משתמשים עשויים להתקבל שגיאה 403 בעת התחברות ל- SharePoint עם SharePoint Designer. עבור שלבים מפורטים, דוגמאות, ראה שיקולי [אפשר או מנע בקובץ script מותאם אישית](https://docs.microsoft.com/sharepoint/allow-or-prevent-custom-script).

אם המשתמשים נתקלים **403 שגיאה** עם SharePoint או OneDrive, באפשרותך לבדוק את [תקינות שירות מחוונים](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) כדי לקבוע אם זוהי בעיה מוכרת עם רזולוציה מתבצע לפני התקשר לתמיכה או לבזבז זמן פתרון בעיות. הבעיה עשויה להיות מאוד זמניות ולא כבר שזוהה על-ידי Microsoft, אך ייתכן שקיימת lapse זמן בין זיהוי בעיה רישום ללוח המחוונים תקינות השירות. נא להמשיך לבדוק את לוח המחוונים של תקינות שירות עדכונים נוספים.
