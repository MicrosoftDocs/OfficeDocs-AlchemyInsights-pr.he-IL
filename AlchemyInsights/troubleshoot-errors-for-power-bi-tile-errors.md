# <a name="troubleshooting-refresh-scenarios---tile-errors"></a>פתרון בעיות בתרחישי רענון-שגיאות אריח

אם אתה נתקל שגיאות רענון אריח בתוך השירות Power BI, נא להמתין כ 10-15 דקות כדי לרענן להיות משתקף באריחי לוח המחוונים. אם הוא עדיין אינו מופיע, הצמד את הפריט החזותי ללוח המחוונים.

למידע נוסף:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
