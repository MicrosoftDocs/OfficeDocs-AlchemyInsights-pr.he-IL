# <a name="activating-windows-10-subscriptions"></a>הפעלת מנויי Windows 10

Microsoft 365 Business, Microsoft 365 השכלה, Microsoft 365 ו-Enterprise מספקים רשיונות דיגיטליים עבור Windows 10 כחלק מהמנוי. כדי להפעיל את המנוי, ההתקן שלך חייב כבר להפעיל את Windows 10 Pro (1703 או גירסה מאוחרת יותר). כדי להציג רשימה מלאה של דרישות להפעלת מנוי של Windows 10, ראה [דרישות הפעלת מנויים](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements)של windows 9.
