# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: פתרון בעיות בעיבוד ' שינויים '

אם אתה רואה את המאמר "עיבוד שינויים" למשך פרק זמן ממושך, ייתכן שהסיבה לכך שקובץ מקוון פתוח, מספר קבצים התקועים בתור או מסנכרן קובץ גדול מאוד, ראה [OneDrive: פתרון בעיות בעיבוד שינויים](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) כדי לסייע בפתרון הבעיה.

בנוסף, תראה הודעה זו אם יש לך קובץ המשתנה כל הזמן (כגון Outlook. קובץ PST או מסד נתונים). למד כיצד [להעביר את קבצי הנתונים של Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) שאתה משתמש בהם באופן קבוע מחוץ לOneDrive.
