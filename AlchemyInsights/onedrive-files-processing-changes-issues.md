# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: פתרון בעיות ' שינויי עיבוד '

אם אתה רואה "עיבוד שינויים" למשך פרק זמן ממושך, זה יכול להיות מכיוון שיש לך קובץ מקוון פתוח, המון קבצים תקועים בתור, או קובץ גדול מאוד סינכרון, ראה [Onedrive תקוע על העיבוד](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd).

קרא מאמר זה כדי לפתור את "עיבוד שינויים". כמו כן, תראה הודעה זו אם יש לך קובץ שמשתנה כל הזמן (כמו Outlook. קובץ PST או מסד נתונים). למד כיצד [להעביר את קבצי הנתונים של Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) שאתה משתמש בהם באופן סדיר מחוץ ל-onedrive.
