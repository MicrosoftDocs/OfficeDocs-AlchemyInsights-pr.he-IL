# <a name="use-whiteboard-with-microsoft-365"></a>שימוש בלוח ציור עם Microsoft 365

Microsoft Whiteboard הוא בד ציור דיגיטלי המיועד לכתיבה ביד חופשית, ומפגיש בין אנשים, תוכן ורעיונות. 

- [הפעל את לוח ציור](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920#bkmk_07) כמנהל מערכת של Microsoft 365. 

- שימוש ב[לוח ציור ב- Microsoft Teams](https://support.microsoft.com/he-IL/office/use-whiteboard-in-microsoft-teams-7a6e7218-e9dc-4ccc-89aa-b1a0bb9c31ee). 

- [שתף פעולה](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920#bkmk_27) עם מישהו אחר בלוח ציור. 

לקבלת מידע נוסף, ראה [עזרת Microsoft Whiteboard](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920).
