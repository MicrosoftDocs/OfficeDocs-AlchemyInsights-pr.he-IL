# <a name="how-to-change-your-theme"></a>כיצד לשנות את הנושא שלך

כדי לשנות את ערכת הנושא ב-Outlook.com או ב-Outlook באינטרנט, עבור אל **הגדרות** ובחר ערכת נושא תחת **ערכת נושא**. כדי לראות את כל ערכות הנושא הזמינות, בחר **בהצג הכל** תחת **ערכת נושא**.
