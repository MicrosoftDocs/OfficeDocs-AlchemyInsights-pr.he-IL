# <a name="use-your-browser-to-check-spelling"></a>השתמש בדפדפן כדי לבדוק איות

אין אפשרות לבדוק איות ב-Outlook באינטרנט. כדי לבדוק את האיות, יהיה עליך להשתמש בדפדפן האינטרנט. בדיקת איות זמינה ב-Microsoft Edge, Internet Explorer 10 וגירסאות מאוחרות יותר וגירסאות נוכחיות של Firefox, Chrome וספארי. בדוק את האפשרויות עבור דפדפן האינטרנט שלך כדי ללמוד עוד אודות אופן בדיקת האיות.
