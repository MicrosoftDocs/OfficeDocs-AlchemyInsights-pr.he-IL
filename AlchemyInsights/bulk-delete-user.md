# <a name="bulk-delete-user"></a>משתמש מחיקה בצובר

באפשרותך להסיר מספר רב של משתמשים מקבוצה באמצעות קובץ ערכים מופרדים באמצעות פסיקים (CSV) כדי למחוק משתמשים בצובר. כדי למחוק משתמשים בצובר המשתמשים בפורטל לספירה, ראה [משתמשים מחיקה בצובר בספריית הפעילות התכלת](https://docs.microsoft.com/azure/active-directory/users-groups-roles/users-bulk-delete).
