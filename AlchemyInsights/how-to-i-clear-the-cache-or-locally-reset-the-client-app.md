# <a name="how-do-i-clear-the-cache-or-locally-resetrefresh-the-clientapp"></a>כיצד ניתן לנקות את המטמון או לאפס/לרענן את הלקוח/app?

אם אתה רואה נתונים לא שלמים או שגויים, נסה לכפות את הסגירה ולפתוח מחדש את האפליקציה.  

אם הדבר אינו פועל, באפשרותך לנסות לאפס את החשבון בהתקן.
 
פתח את היישום, עבור אל **הגדרות**, בחר את חשבון הדואר האלקטרוני, ובחר **איפוס חשבון**.
