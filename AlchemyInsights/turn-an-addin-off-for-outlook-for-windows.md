# <a name="turn-an-add-in-off-for-outlook-for-windows"></a>הפעלה של תוספת עבור Outlook עבור Windows

אם אינך מעוניין להשתמש בתוספת ב-Outlook 2013 או ב-Outlook 2016 עבור Windows, באפשרותך להפוך אותה ללא זמינה כך שלא תופיע בהודעות שלך.  

1. ב-Outlook, לחץ על **File _GT_ ניהול תוספות**.
2. לחץ על ' **נהל תוספות** '
3. עצה: פעולה זו פותחת את Outlook באינטרנט.
4. תחת ניהול תוספות, בעמודה מופעל, **בטל את הבחירה בתיבה עבור התוספת**  שברצונך לבטל.
5. הפעלה או ביטול של תוספות

עצה: כדי להפעיל את התוספת בחזרה, סמן את אותה תיבה.
