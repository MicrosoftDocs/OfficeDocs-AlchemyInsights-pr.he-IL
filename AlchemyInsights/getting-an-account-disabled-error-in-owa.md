# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>מקבל שגיאת חשבון שאינה זמינה ב-Outlook באינטרנט?

אם אתה מקבל את השגיאה **שהחשבון שלך הפך ללא זמין**  `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` כאשר אתה מנסה לפתוח את Outlook באינטרנט (שנקרא בעבר outlook web App), ייתכן שמנהל הארגון שלך הפך את הגישה שלך ל-outlook באינטרנט. מנהל המערכת יכול לתקן שגיאה זו על-ידי ביצוע השלבים בנושא [הפיכת Outlook Web App לזמין או ללא זמין עבור תיבת דואר](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
