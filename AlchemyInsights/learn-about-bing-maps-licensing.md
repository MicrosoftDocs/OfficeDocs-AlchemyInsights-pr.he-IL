# <a name="learn-about-bing-maps-licensing"></a>למד אודות רישוי מפות Bing

השירות ' מפות Bing ' מציע כמה אפשרויות רישוי גמיש שיתמכו בצרכים ובדרישות הטכניות של הארגון שלך. כמו כן, באמצעות הרמה ללא תשלום, באפשרותך להתחיל את הבנייה עם מפות Bing היום.

לקבלת פרטים, ראה [רישוי מפות של Bing](https://go.microsoft.com/fwlink/?linkid=2150203).
