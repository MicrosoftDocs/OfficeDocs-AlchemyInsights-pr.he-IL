# <a name="got-a-parameter-cannot-be-found-error"></a>האם לא ניתן למצוא את הפרמטר ' שגיאה '?

אם אתה מקבל את הודעת השגיאה **"אין אפשרות למצוא פרמטר התואם לשם פרמטר ' EmailAddresses"**, משמעות הדבר היא שלוקח זמן רב יותר עבור Microsoft 365 לסיים את הגדרת הדייר שלך, או את התחום המותאם אישית שלך אם הוספת לאחרונה אחד.
  
תהליך ההתקנה יכול להימשך עד 4 שעות להשלמת הפעילות. המתן זמן רב כדי שתהליך ההגדרה יהיה מועד לסיום ולאחר מכן נסה שוב.
