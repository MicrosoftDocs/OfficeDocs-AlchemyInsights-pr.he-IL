# <a name="accessibility-features-in-microsoft-edge"></a>תכונות נגישות ב-Microsoft Edge

Microsoft Edge מגיע עם אפשרויות נגישות מוכללות שיסייעו לך לגלוש באינטרנט באופן המתאים ביותר לצרכיך.

לקבלת מידע נוסף, ראה [תכונות נגישות ב-Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2153648).
