# <a name="learn-about-microsoft-365-groups"></a>קבלת מידע אודות קבוצות Microsoft 365

קבוצות Microsoft 365 היא שירות הפועל עם כלי Office 365 שבהם אתה משתמש כבר, כך שתוכל לשתף פעולה עם חברי הצוות שלך בעת כתיבת מסמכים, יצירת גליונות אלקטרוניים, עבודה על תוכניות פרוייקט, תזמון פגישות או שליחת דואר אלקטרוני. לקבלת מידע נוסף, ראה קבלת מידע [אודות קבוצות Microsoft 365](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
