# <a name="subscription-recurring-billing"></a>חיוב חוזרת של מנוי

אם ברצונך לדעת מתי שלך מנוי חודשי או שנתי תחויב (במקרה כזה **מחזורי חיוב** ) או פקיעת תוקף (אם **מחזורי חיוב** כבויה):
  
1. במרכז הניהול, עבור לדף [מנויים](https://go.microsoft.com/fwlink/p/?linkid=842054), או בחר **חיוב** \> **מנויים**.

2. בחר את המנוי שברצונך לדעת יותר אודות.

3. בצד ימין של כרטיס המנוי הוא מידע אודות מצב מנוי, ההגדרה **מחזורי חיוב לפי** תאריך חיוב או התפוגה הבא.

4. כדי להמשיך להשתמש בשירות שתאריך זה, לעבור את מצבי **הגדרות החיוב של מחזורי** **בסרגל**. המצב תציג תאריך החיוב הבא עבור המנוי.

5. כדי להפסיק את שירות בתאריך זה, לעבור מצבי **מחזורי חיוב** **ביטול**. המצב יציג את התאריך שפג תוקפו של המנוי.
