# <a name="create-custom-microsoft-surface-operating-system-images"></a>יצירת תמונות מותאמות אישית של מערכת ההפעלה Microsoft Surface

השתמש במאיץ הפריסה (SDA) של Microsoft Surface כדי לפשט ו להפוך את הפריסה של מערכת הפעלה חדשה (OS) לאוטומטית, המותאמה לאוגר שלך ומותאמת להמלצות Microsoft.

1. ב- GitHub, עבור אל SurfaceDeploymentAccelerator.

1. בחר **קוד** ולאחר מכן בחר **פתח באמצעות GitHub Desktop או** הורד **ZIP**.

1. סקור את **README.md כדי** ללמוד כיצד פועל קובץ ה- Script.

1. ערוך את קובץ ה- Script עם המשתנים המתאימים עבור הסביבה שלך כפי שתועד בקובץ Readme ולאחר מכן סקור אותו לפני הפעלתו בסביבת מחשב שלך.
