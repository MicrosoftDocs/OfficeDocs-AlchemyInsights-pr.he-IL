# <a name="site-is-not-available"></a>האתר אינו זמין

אם אתר SharePoint או OneDrive שלך אינו זמין למשתמשים מרובים, ייתכן שקיימת בעיית שירות זמנית. בדוק את [לוח המחוונים של השירות](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) כדי לראות אם הדייר שלך מושפע.
