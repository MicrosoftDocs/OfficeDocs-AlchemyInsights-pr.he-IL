# <a name="search-experience-updated"></a>חוויית חיפוש מעודכן

חיפוש הוא כעת הממוקם בחלקו התחתון של היישום. הקשה עליו מעניק לך גישה ישירה שלך אנשי קשר עליון, לטיולים עתידיים, משלוחים חבילת וקבצים מצורפים אחרונים. באפשרותך לחפש הודעות דואר אלקטרוני, קבצים ואנשי קשר בכל חשבונותיך מתיבת החיפוש יחידה אחת או להשתמש במסננים החדש שלנו כדי לצמצם את החיפוש.
