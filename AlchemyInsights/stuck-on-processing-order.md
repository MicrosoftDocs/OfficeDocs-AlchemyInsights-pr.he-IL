# <a name="stuck-on-processing-order"></a>תקוע על עיבוד הזמנה

אם בחרת לשדרג את מנוי Microsoft 365 שלך, אבל ההזמנה נתקעה על “מעבד את ההזמנה שלך“, פנה אל פרסום  התהליך של שדרוג הדייר שלך ידרוש הקצאת משאבים, שעשויים להתעכב. אנחנו צופים שהתהליך יושלם תוך 24 שעות; עם זאת, ראינו מקרים נדירים שבהם בקשות אלו לקחו עד 48 שעות. פתח בקשת שירות אם עיבוד ההזמנה נמשך למעלה מ-48 שעות.
