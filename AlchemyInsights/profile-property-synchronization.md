# <a name="profile-property-synchronization"></a>סינכרון מאפיין פרופיל

אין אפשרות למפות מאפיינים נוספים עבור ייבוא AD לסינכרון מ- Active Directory תכלת הרקיע אל יישום פרופיל המשתמש. ייבוא AD מוגבלת שנקבעו מראש עבור ערכת מאפיינים כדי להבטיח ביצועים עקביים של משימת שעון העצר.
