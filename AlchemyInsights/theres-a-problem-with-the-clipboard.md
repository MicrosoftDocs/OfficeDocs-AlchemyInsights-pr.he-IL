# <a name="resolving-excel-clipboard-error"></a>פתרון שגיאת לוח Excel

אם אתה מקבל שגיאה ב- Excel שאומרת, "קיימת בעיה בלוח, אך עדיין תוכל להדביק את התוכן שלך בתוך חוברת עבודה זו", נסה להפעיל את התכונה Live Preview על-ידי ביצוע השלבים הבאים:

1. ב- Excel, לחץ **על אפשרויות**  >  **קובץ.**
3. בתיבת **הדו-שיח אפשרויות Excel,** בחר **כללי**.
4. תחת **אפשרויות ממשק משתמש,** נקה את הפוך **תצוגה מקדימה חיה לזמינה.**
5. לחץ על **אישור**.
6. הפעל מחדש את Excel.
