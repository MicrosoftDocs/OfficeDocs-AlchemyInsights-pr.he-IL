# <a name="export-user-account-information"></a>ייצוא מידע של חשבון משתמש

כדי לייצא מידע של חשבון משתמש במרכז הניהול של Microsoft 365, כולל מידע אודות רשיונות, מדיניות סיסמה, חשבונות חסומים, מצב סינכרון של הרשאות ועוד, עבור אל הדף **משתמשים**  >  [פעילים](https://go.microsoft.com/fwlink/p/?linkid=834822) ולאחר מכן בחר **יצא משתמשים**.
