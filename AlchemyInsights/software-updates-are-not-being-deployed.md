# <a name="software-updates-are-not-being-deployed"></a>עדכוני תוכנה אינם נפרסים

אם קבעת את תצורת עדכוני התוכנה באמצעות טבעת עדכון של Windows 10, אך העדכונים אינם נפרסים, נסה את הפעולות הבאות:  

- שקול לשנות את שירות השירות של Windows מסוג ההפצה  **החצי שנתי**  לסוג הפצה מחמיר ותדיר יותר.
- בדוק את התקופה הדחייה עבור עדכון  **איכות**  **ועדכון תכונות**. תקופת דחייה עלולה לגרום לעיכוב בעדכונים עד 180 ימים.
