# <a name="unable-to-print"></a>אין אפשרות להדפיס

אנחנו מודעים בעיה שבה אין אפשרות להדפיס חלק מהמשתמשים בעת שימוש בדפדפנים Safari או קצה. כאשר אנו עובדים על תיקון זה, נא השתמש בדפדפן אחר. אנו מתנצלים אי הנוחות.
