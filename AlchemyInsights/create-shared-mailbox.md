# <a name="create-a-shared-mailbox"></a>יצירת תיבת דואר משותפת

תיבת דואר משותפת מאפשרת ליותר מאדם אחד שליחה וקבלה של דואר אלקטרוני עבור כינוי, כמו sales@contoso.com, מבלי להקצות רשיון נוסף. כדי ליצור תיבת דואר משותפת, עבור אל [מדריך דוא ל משותף](https://portal.office.com/adminportal/home).
  
עיין במאמר [יצירת תיבת דואר משותפת](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) עבור עצות ופתרונות מידע נוסף אודות השימוש בתיבת דואר משותפת.
