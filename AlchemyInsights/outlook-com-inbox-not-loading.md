# <a name="inbox-not-loading-with-firefox"></a>תיבת הדואר הנכנס לא טוען עם Firefox

הרחבות מסוימות בדפדפן Firefox עלול למנוע Outlook.com inbox טעינה.
  
בקר תמיכה Firefox כדי ללמוד כיצד [להפוך ללא זמין או להסיר הרחבות](https://support.mozilla.org/kb/disable-or-remove-add-ons).
