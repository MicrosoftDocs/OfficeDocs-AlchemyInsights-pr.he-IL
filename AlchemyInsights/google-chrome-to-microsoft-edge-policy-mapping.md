# <a name="learn-about-mapping-between-google-chrome-policies-and-microsoft-edge-policies"></a>קבלת מידע אודות מיפוי בין מדיניות Google Chrome ומדיניות של Microsoft Edge

מדיניות Google Chrome ממופה למדיניות הקצה הרלוונטית של Microsoft הנתמכת בגירסה 80 של הדפדפן. לקבלת מידע נוסף, ראה [Google Chrome למיפוי מדיניות קצה של Microsoft](https://go.microsoft.com/fwlink/?linkid=2141933).
