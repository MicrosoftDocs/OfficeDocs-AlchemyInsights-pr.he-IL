# <a name="missing-localizednames-in-element-savedquery"></a>חסר ' LocalizedNames ' ברכיב ' savedquery '

לקבלת עזרה בנושא ' LocalizedNames ', ראה ["הרכיב ' savedquery ' כולל תוכן לא שלם. רשימת הרכיבים האפשריים הצפויים: השגיאה ' LocalizedNames ' מתרחשת בעת ייבוא פתרון של Dynamics 365](https://support.microsoft.com/help/4463330/the-element-savedquery-has-incomplete-content-list-of-possible-element).

חפש תווים מיוחדים בשם התצוגה שעשוי לגרום לבעיה. הסר את התווים המיוחדים שם התצוגה ויצא את הפתרון שוב.
