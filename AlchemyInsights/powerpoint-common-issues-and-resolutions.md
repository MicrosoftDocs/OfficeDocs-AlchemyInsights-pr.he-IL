# <a name="powerpoint-common-issues-and-resolutions"></a>בעיות נפוצות ב-PowerPoint ורזולוציות

להלן כמה נושאים המבוקשים לעתים קרובות ב-PowerPoint:

1. [PowerPoint אינו מגיב, נתקע או קופא](https://support.office.com/article/PowerPoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d).

2. [ל-PowerPoint אין אפשרות להוסיף וידאו מתוך הקובץ שנבחר](https://support.office.com/article/PowerPoint-cannot-insert-a-video-from-the-selected-file-acd46430-9e0c-4dca-9484-19cf0afdde7c).

3. **סרטוני וידאו**: [הוספת קובץ וידאו מהמחשב שלך](https://support.office.com/article/insert-and-play-a-video-file-from-your-computer-f3fcbd3e-5f86-4320-8aea-31bff480ed02), [הוספת וידאו מ-YouTube](https://support.office.com/article/Insert-a-video-from-YouTube-or-another-site-8340ec69-4cee-4fe1-ab96-4849154bc6db), [ותבניות קבצי וידאו ושמע שנתמכות ב-PowerPoint](https://support.office.com/article/video-and-audio-file-formats-supported-in-powerpoint-d8b12450-26db-4c7b-a5c1-593d3418fb59).

4. **הדפסה**: לקבלת שיטות עבודה מומלצות ובעיות הקשורות להדפסה, סקור את המידע [בהדפסת שקופיות והערות של PowerPoint](https://support.office.com/article/Print-your-PowerPoint-slides-handouts-or-notes-194d4320-aa03-478b-9300-df25f0d15dc4). 

5. **גופנים**: כדי לשתף גופנים עם המצגות שלך, קרא [גופנים מוטבעים ב-Word או ב-PowerPoint](https://support.office.com/article/Embed-fonts-in-Word-or-PowerPoint-cb3982aa-ea76-4323-b008-86670f222dbc).

6. **תכונות**: [השוואה בין תכונות PowerPoint](https://support.office.com/article/Compare-PowerPoint-features-on-different-platforms-90986850-227c-4b25-938e-1c5838166b8b#bm11) בפלטפורמות Windows, Mac, web ו-mobile.
