# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>התקנה, התקנה, ולסנכרן קבצים OneDrive או SharePoint ב- Mac 

התקן, הגדרת סינכרון OneDrive או SharePoint וקבצי ב- Mac עם ההוראות מקבצי [סינכרון עם הלקוח OneDrive הסינכרון ב- Mac OS X](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f).

אם התקנת את יישום OneDrive ההתקנה ב- Mac, אך משתמשים נתקלים בבעיות סינכרון, ראה [סינכרון תיקון בעיות OneDrive (Mac)](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756?ui=en-US&amp;rs=en-US&amp;ad=US).

> [!NOTE]
> תכונה ידועה להעביר תיקיה בסינכרון OneDrive עבור Windows אינו זמין עבור mac.
