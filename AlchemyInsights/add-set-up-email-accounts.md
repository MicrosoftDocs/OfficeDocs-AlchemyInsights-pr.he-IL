# <a name="problems-setting-up-an-email-account-in-outlook"></a>בעיות בהגדרת חשבון דואר אלקטרוני ב-Outlook

אם אתה נתקל בבעיות בהגדרת חשבון Exchange ב-Outlook, באפשרותך להפעיל את [כלי ההגדרה של חשבון Outlook](https://aka.ms/SaRA-OutlookSetupProfile) במחשב המושפע. היא תבצע בדיקות שונות ותספק פתרונות אפשריים לבעיית הגדרת החשבון שלך.
  
אם אתה עדיין נתקל בבעיות, ראה [Outlook אינו יכול להגדיר פרופיל חדש באמצעות גילוי אוטומטי של exchange עבור תיבת דואר של Exchange Online](https://docs.microsoft.com/exchange/troubleshoot/outlook-profiles/cannot-set-up-profile-autodiscover).
