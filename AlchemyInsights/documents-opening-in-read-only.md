# <a name="documents-opening-in-read-only"></a>מסמכים שנפתחים לקריאה בלבד

ייתכן שתמצא את הקבצים שאתה פותח לקריאה בלבד. במקרים מסוימים, תנאי זה משמש כאבטחה שנוספה, כגון בעת פתיחת קבצים מהאינטרנט ושעות אחרות, מצב זה עשוי לנבוע מהגדרה הניתנת לשינוי.

- [קבצי SharePoint נפתחים לקריאה בלבד](https://docs.microsoft.com/sharepoint/troubleshoot/lists-and-libraries/files-open-as-read-only-and-cannot-check-in-or-out)

אם הלקוח מאמין שהסיבות המוצהרות, כגון מנגנון שיפור האבטחה או הגדרה ללא שינוי, אינם אחד מהתרחישים המפורטים לעיל [בקבצי SharePoint הנפתחים לקריאה בלבד](https://docs.microsoft.com/sharepoint/troubleshoot/lists-and-libraries/files-open-as-read-only-and-cannot-check-in-or-out), להלן פתרונות נפוצים אחרים למסמכים הנפתחים במצב קריאה בלבד.

- הפעל מחדש את המחשב.
- התקנת עדכוני Office.
- בדוק את האנטי-וירוס במחשב.
- ודא שהאתר נמצא באזור ' אתרים מהימנים ' ב-Internet Explorer.
- ביצוע תיקון של Office.
- נקה את [מטמון מסמכי Office](https://support.microsoft.com/office/delete-your-office-document-cache-b1d3765e-d71b-4bb8-99ca-acd22c42995d?ui=en-us&rs=en-us&ad=us).
