# <a name="uk-domain-transfers"></a>העברות תחום בבריטניה

ontact של Microsoft Support ובקש **שינוי תג IPS** כדי להתאים לרשם שברצונך לנהל את התחום שלך ולהעביר אותו קדימה. לאחר שהתג משתנה, התחום מועבר באופן מיידי לרשם החדש. לאחר מכן תצטרך לעבוד עם הרשם החדש כדי להשלים את ההעברה, כנראה לשלם דמי העברה ולהוסיף את התחום שהועבר לחשבון שלך באמצעות הרשם החדש.

לקבלת מידע נוסף אודות העברות תחום, ראה [העברת תחום מ-Microsoft למארח אחר](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/transfer-a-domain-from-microsoft-to-another-host?view=o365-worldwide).
