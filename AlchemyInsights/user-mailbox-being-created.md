# <a name="your-users-mailbox-is-being-created"></a>תיבת הדואר של המשתמש שלך נוצרת

באפשרותך להימשך עד שעה עד שתיבת דואר חדשה תהיה מוכנה לשימוש.
  
אם תיבת הדואר של המשתמש "הכנה" במשך יותר משעה, באפשרותך לנסות למחוק וליצור מחדש את המשתמש, או להשתמש באפשרות ' תן לנו להתקשר אליך ' כדי לשוחח עם מהנדס תמיכה.
