# <a name="your-users-mailbox-is-being-created"></a>יצירת תיבת דואר של משתמש

הוא יכול להימשך עד שעה לפני לתיבת דואר חדשה מוכן לשימוש.
  
אם תיבת הדואר של המשתמש יש עבר "להכנת" יותר משעה, באפשרותך לנסות למחוק יצירה מחדש של המשתמש, או השתמש באפשרות "הודע לנו להתקשר אליך" לשוחח עם מהנדס תמיכה.
