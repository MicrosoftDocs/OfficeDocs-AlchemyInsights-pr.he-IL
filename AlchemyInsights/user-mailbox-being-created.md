# <a name="your-users-mailbox-is-being-created"></a>תיבת הדואר של המשתמש שלך נוצרת

הפעולה יכולה להימשך עד שעה לפני שתיבת דואר חדשה מוכנה לשימוש.
  
אם תיבת הדואר של המשתמש "הכינה" במשך יותר משעה, באפשרותך לנסות למחוק וליצור מחדש את המשתמש, או להשתמש באפשרות "תן לנו לקרוא לך" כדי לדבר עם מהנדס תמיכה.
