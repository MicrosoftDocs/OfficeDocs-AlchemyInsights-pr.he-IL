# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>פתרון בעיות התקנת טלפון נייד של Outlook עבור תיבת דואר משותפת

אין אפשרות לגשת תיבת הדואר המשותפת מתוך יישום Outlook המותקנת במכשיר הנייד. האם ברצונך לאפשר תכונה זו? ליידע את צוות ההנדסה של Outlook יודע. [ההצבעה כאן](https://go.microsoft.com/fwlink/?linked=862116)!
  
באפשרותך להשתמש רק תיבת דואר משותפת במכשיר נייד על-ידי פתיחת הדפדפן שלך, הכניסה אל Office 365 ולאחר מכן עבור Outlook באינטרנט. לקבלת הוראות, ראה [הוספת תיבת דואר משותפת ב- Outlook באינטרנט](https://support.office.com/en-us/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). מ- Outlook באינטרנט יהיה באפשרותך לגשת לתיבת הדואר המשותפת.
