# <a name="credit-request-due-to-a-service-outage"></a>בקשת אשראי עקב הפסקת שירות

כדי להגיש בקשה לזיכוי עקב הפסקת שירות, פתח בקשת שירות.

הקפד לקבל את מזהה האירוע, נמצא [בדף תקינות השירות](https://docs.microsoft.com/office365/enterprise/view-service-health) במרכז הניהול של Microsoft 365.

לידיעתך, מאחר שהבקשה שלך מעובדת, היא תעבור בתוקף כדי לבחון את ההשפעה של הפסקת החשמל.
