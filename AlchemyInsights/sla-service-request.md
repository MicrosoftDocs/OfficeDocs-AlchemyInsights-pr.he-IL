# <a name="credit-request-due-to-a-service-outage"></a>בקשת זיכוי עקב הפסקת שירות

כדי להגיש בקשה לזיכוי עקב הפסקת שירות, נא פתח בקשת שירות.

נא הקפד לקבל את מזהה האירוע, נמצא [בדף הבריאות של שירות](https://docs.microsoft.com/office365/enterprise/view-service-health) במרכז הניהול של Microsoft 365.

לידיעתך, כפי שהבקשה שלך מעובדת, זה יהיה לעבור אימותים כדי לבחון את ההשפעה של הפסקת.
