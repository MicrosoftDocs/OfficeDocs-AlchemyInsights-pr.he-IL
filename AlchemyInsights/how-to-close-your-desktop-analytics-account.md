# <a name="how-to-close-your-desktop-analytics-account"></a>כיצד לסגור חשבון Analytics שולחן העבודה שלך

אם הגדרת Analytics שולחן עבודה בסביבה שלך, ולאחר מכן תחליט שעליך להסיר אותו, השתמש התהליך בקישור הבא:

[כיצד לסגור את החשבון שלך](https://docs.microsoft.com/sccm/desktop-analytics/account-close)

השלב הראשון הוא ליצור קשר עם התמיכה של Microsoft. פתח אירוע תמיכה כדי לסגור את החשבון Analytics בשולחן העבודה שלך. אין להמשיך עם שלבים נוספים עד שתקבל אישור Microsoft לסגור את החשבון שלך.
