# <a name="are-you-looking-for-phones-that-are-compatible-with-teams"></a>האם אתה מחפש טלפונים התואמים לצוותים?

כמנהל, אתה [מנהל את כל ההתקנים](https://docs.microsoft.com/microsoftteams/device-management) הנמצאים בשימוש עם צוותים בארגון שלך ממרכז הניהול של Microsoft 365. למידע נוסף: 

- סקור את רשימת [הטלפונים עבור צוותי Microsoft](https://docs.microsoft.com/microsoftteams/phones-for-teams). 
- סקור את רשימת [מכשירי ה-IP המאושרים כרגע](https://docs.microsoft.com/microsoftteams/teams-ip-phones#currently-certified-ip-phones). 
- סקור את רשימת [ההתקנים בעלי מגבלות ובעיות ידועות](https://support.office.com/article/control-calls-using-a-headset-in-teams-65d6e104-444d-4013-b8c2-f11317dd69a8).
