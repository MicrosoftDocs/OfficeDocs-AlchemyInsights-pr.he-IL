# <a name="gatewaynotreachable-when-setting-credentials"></a>GatewayNotReachable בעת הגדרת אישורים

אתה עשוי להיתקל GatewayNotReachable בעת ניסיון להגדיר את האישורים עבור מקור נתונים. הדבר עשוי להיות התוצאה של שער מיושנים. התקנת השער העדכני ביותר ולאחר מכן נסה שוב.
