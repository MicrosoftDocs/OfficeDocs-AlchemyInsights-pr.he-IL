# <a name="your-users-mailbox-is-being-created"></a>תיבת הדואר של המשתמש שלך נוצרת

באפשרותך להימשך עד שעה עד שתיבת דואר חדשה תהיה מוכנה לשימוש.
  
אם תיבת דואר של משתמש כבר "הכנה" במשך יותר משעה, באפשרותך לנסות למחוק ולאחר מכן ליצור מחדש את המשתמש, או להשתמש באפשרות ' תן לנו להתקשר אליך ' כדי לשוחח עם מהנדס תמיכה.
