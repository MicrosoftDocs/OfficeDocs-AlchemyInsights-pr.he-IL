# <a name="spam-filter-not-working"></a>מסנן דואר זבל אינו פועל

אם אתה סבור שהודעה שהתקבלה ב-Outlook.com היא הונאת דיוג או זיוף, בחר את ההודעה ובחר באפשרות **דיוג** **זבל** \> כדי לסמן אותה כתרמית דיוג. כדי ללמוד עוד אודות התעללות, דיוג והתחזות, [קרא את המאמר](https://support.office.com/article/0d882ea5-eedc-4bed-aebc-079ffa1105a3?wt.mc_id=Office_Outlook_com_Alchemy). כדי לחסום כתובות דואר אלקטרוני, הוסף כתובות או תחומים לרשימת השולחים הבטוחים או החל אפשרויות דואר זבל אחרות, בחר **באפשרויות דואר זבל**.
