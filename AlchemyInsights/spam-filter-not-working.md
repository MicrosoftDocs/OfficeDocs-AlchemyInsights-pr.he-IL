# <a name="spam-filter-not-working"></a>מסנן הודעות זבל אינו פועל

אם אתה מאמין שהודעה שקיבלת ב-Outlook.com היא הונאת דיוג או התחזות, בחר את ההודעה **Junk** ובחר \> **דיוג** זבל כדי לסמן אותה כתרמית דיוג. לקבלת מידע נוסף על התעללות, דיוג והתחזות, [קרא את המאמר](https://support.office.com/article/0d882ea5-eedc-4bed-aebc-079ffa1105a3?wt.mc_id=Office_Outlook_com_Alchemy). כדי לחסום כתובות דואר אלקטרוני, להוסיף כתובות או תחומים לרשימת השולחים הבטוחים, או להחיל אפשרויות דואר זבל אחרות, בחר **אפשרויות דואר זבל**.
