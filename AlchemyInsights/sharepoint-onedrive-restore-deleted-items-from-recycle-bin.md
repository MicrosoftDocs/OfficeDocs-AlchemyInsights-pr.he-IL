# <a name="restore-a-deleted-item"></a>שחזור פריט שנמחק

[בדוק את סל המיחזור של האתר](https://support.office.com/article/restore-deleted-items-from-the-site-collection-recycle-bin-5fa924ee-16d7-487b-9a0a-021b9062d14b?ui=en-US&amp;rs=en-US&amp;ad=US)

[שחזור פריטים בסל המיחזור של אתר SharePoint](https://support.office.com/article/Restore-deleted-files-or-folders-in-OneDrive-949ada80-0026-4db3-a953-c99083e6a84f)



התכונה שחזור קבצים

אם רבים מהקבצים שלך OneDrive או SharePoint לקבל נמחקים, מוחלפים, פגום או נגוע על-ידי תוכנות זדוניות, באפשרותך לשחזר את הספריה כולה OneDrive או Sharepoint לזמן קודם באמצעות תכונת שחזור קבצים.

[שחזור ספריה OneDrive](https://support.office.com/article/restore-your-onedrive-fa231298-759d-41cf-bcd0-25ac53eb8a15)

[שחזור ספריית מסמכים](https://support.office.com/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a?ui=en-US&amp;rs=en-US&amp;ad=US)

השתמש רישום ביקורת או בחלונית הפעילות קובץ כדי לבדוק את ההיסטוריה של הקובץ.

[בדוק את דוחות ביקורת](https://docs.microsoft.com/office365/securitycompliance/search-the-audit-log-in-security-and-compliance?redirectSourcePath=%252fen-us%252farticle%252fsearch-the-audit-log-in-the-office-365-protection-center-0d4d0f35-390b-4518-800e-0c7ec95e946c) </a> על-ידי ניווט [כאן](https://protection.office.com/#/unifiedauditlog)

השתמש בחלונית [הפעילות קובץ](https://support.office.com/article/File-activity-in-a-document-library-6105ecda-1dd0-4f6f-9542-102bf5c0ffe0) כדי לבדוק את ההיסטוריה של הקובץ.

בדוק את הלקוח סינכרון OneDrive במחשב המקומי.  אם אתה מסנכרן את הקבצים במחשב שלך דרך לקוח OneDrive הסינכרון, בדוק את התיקיה סינכרון מקומי כדי להבטיח כי אותו כראוי הועלה. ודא שגם לבדוק את סל המיחזור במחשב המקומי.
