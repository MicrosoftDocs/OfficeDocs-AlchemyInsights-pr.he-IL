# <a name="submit-a-repair-request-for-a-surface-or-xbox-device"></a>שליחת בקשת תיקון עבור מכשיר משטח או מכשיר Xbox

כדי לשלוח בקשה לתיקון או להחלפה עבור המכשיר שלך, בצע את הפעולות הבאות:

1. עבור אל **שירותי המכשירים** והיכנס באמצעות חשבון Microsoft שלך.

2. בחר את המכשיר שאליו ברצונך לשלוח בקשה לשירות.

3. בחר **התחל הזמנת שירות**, ובצע את ההוראות כדי להשלים את תהליך השליחה.

**הערה** יש לרשום את המכשיר שלך לפני שתוכל לבדוק את האחריות שלך או לקבל שירות. אם עליך לרשום את המכשיר שלך או לא לראות את המכשיר שלך מופיע לאחר שנכנסת לחשבון שלך, עבור אל [רישום המשטח או Xbox](https://support.microsoft.com/surface/register-your-surface-or-xbox-fd7d73f8-b0e6-c9fa-e83b-0b64652e2376).
