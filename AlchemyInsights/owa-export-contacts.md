# <a name="export-contacts"></a>ייצוא אנשי קשר

1. בחלק התחתון של הדף, בחר בסמל אנשים כדי לעבור אל הדף ' אנשים.
2. בסרגל הכלים, בחר **ניהול** > **לייצא אנשי קשר**. 
3. לבחור לייצא את כל אנשי הקשר או אנשי קשר בלבד מתוך תיקיה מסוימת, ולאחר מכן בחר **לייצא**.
4. בחלק התחתון של הדף, בחר **שמור** כדי לשמור את "contacts. csv" תיקיית הורדות ברירת המחדל שלך. לחלופין, בחר על החץ לצד **שמור** ובחר באפשרות **שמירה בשם**.

לקבלת מידע נוסף, ראה [ייצוא אנשי קשר מ- Outlook](https://support.office.com/article/10f09abd-643c-4495-bb80-543714eca73f#ID0EAACAAA=Outlook_on_the_web).
