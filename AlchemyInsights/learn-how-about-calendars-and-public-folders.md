# <a name="learn-about-using-public-folders-in-outlook"></a>קבלת מידע אודות שימוש בתיקיות ציבוריות ב-Outlook

תיקיות ציבוריות ב-Outlook מעניקות לך מקום נהדר לאיסוף, ארגון ושיתוף של מידע אודות נושאים או פרוייקטים מסוימים בתוך הארגון שלך. לקבלת מידע נוסף, ראה [יצירה ושיתוף של תיקיה ציבורית ב-Outlook](https://support.office.com/article/a2835011-d524-4a5c-a207-05c159bb2a97).
