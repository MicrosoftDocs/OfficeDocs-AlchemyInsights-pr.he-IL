# <a name="learn-about-using-public-folders-in-outlook"></a>למד אודות שימוש בתיקיות ציבוריות ב-Outlook

תיקיות ציבוריות ב-Outlook מעניקים לך מקום מצוין לאיסוף, ארגון ושיתוף מידע אודות נושאים או פרוייקטים מסוימים בארגון שלך. לקבלת מידע נוסף, ראה [יצירה ושיתוף של תיקיה ציבורית ב-Outlook](https://support.office.com/article/a2835011-d524-4a5c-a207-05c159bb2a97).
