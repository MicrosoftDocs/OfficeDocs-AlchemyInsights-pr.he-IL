# <a name="set-up-microsoft-information-protection"></a>הגדרת הגנת מידע של Microsoft

קבל מבט כולל על היכולות שניתן להחיל על אסטרטגיית הגנת המידע שלך כדי שתוכל להיות בטוח שהמידע הרגיש שלך מוגן. השתמש בגישת מחזור חיים של ארבע השלבים שבה אתה מגלה, מסווג, מגונן ומנטר מידע רגיש. מדריך "ההגדרה להגנת מידע של Microsoft" מספק הדרכה להשלמת כל אחד משלבים אלה.
