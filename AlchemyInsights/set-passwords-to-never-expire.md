# <a name="set-passwords-to-never-expire"></a>להגדיר סיסמאות יפוג לעולם

הגדרת סיסמאות יפוג לעולם מתוך [אבטחה &amp; פרטיות](https://portal.office.com/adminportal/home#/settings/security). לחץ על "ערוך" לצד מדיניות סיסמה, והפעל את המחוון כדי "פעיל".
  
לקבלת מידע מפורט יותר, ראה: [הגדרת מדיניות תפוגה של סיסמה עבור הארגון שלך](https://support.office.com/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f).
