# <a name="sharepoint-migration-throttling"></a>ויסות העברה של SharePoint

SharePoint Online משתמש בוויסות כדי לשמור על ביצועים ומהימנות אופטימליים של שירות SharePoint Online. ויסות מגביל את מספר פעולות המשתמש או מספר השיחות בו-זמנית (לפי script או קוד) כדי למנוע שימוש מופרז במשאבים. לקבלת מידע נוסף, אנא ראה:

- [אני חווה ביצועים ירודים או ויסות במהלך העברה](https://docs.microsoft.com/sharepointmigration/sharepoint-online-and-onedrive-migration-speed#faq-and-troubleshooting)
