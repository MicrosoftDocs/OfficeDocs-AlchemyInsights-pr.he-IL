# <a name="address-book-in-outlookcom"></a>פנקס הכתובות ב- Outlook.com

פנקס הכתובות שלך Outlook.com נקרא אנשים.

כדי לעבור לדף [אנשים](https://outlook.live.com/people/) שלך, בחר **אנשים** בפינה התחתונה של העמוד, או מהבורר app בפינה העליונה של הדף.

למד כיצד [יצירה, תצוגה, ואנשי עריכה ב- Outlook.com](https://support.office.com/article/5b909158-036e-4820-92f7-2a27f57b9f01).
