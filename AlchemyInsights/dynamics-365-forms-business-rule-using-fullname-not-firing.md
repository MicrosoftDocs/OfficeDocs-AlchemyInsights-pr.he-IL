# <a name="full-name-field-not-supported-with-unified-inteface-apps"></a>אין תמיכה בשדה ' שם מלא ' עם יישומים מאוחדים של הפנים

פעולות או תנאים המשתמשים בשדה ' **שם מלא** ' (Full name) אינם נתמכים ביישומים המבוססים על הממשק האחיד. לחלופין, באפשרותך להשתמש בפעולות או בתנאים עם שדות **שם פרטי (שם** פרטי) ושם **משפחה (שם** משפחה).
