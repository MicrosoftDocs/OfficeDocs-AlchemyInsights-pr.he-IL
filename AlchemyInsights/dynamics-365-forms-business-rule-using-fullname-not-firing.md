# <a name="full-name-field-not-supported-with-unified-inteface-apps"></a>השדה ' שם מלא ' אינו נתמך עם יישומי inteface מאוחדים

פעולות או תנאים המשתמשים בשדה ' **שם מלא** ' (fullname) אינם נתמכים ביישומים המבוססים על הממשק המאוחד. לחלופין, באפשרותך להשתמש בפעולות או בתנאים עם שדות **שם פרטי** (firstname) ושם **משפחה** (lastname).
