# <a name="cannot-add-or-create-a-new-signature-in-outlook"></a>אין אפשרות להוסיף או ליצור חתימה חדשה ב-Outlook

זוהי בעיה ידועה. אנו עובדים כדי לתקן אותה והמצב יעודכן ב [-Outlook קופא בלחיצה על לחצן חתימה](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02).

כדי לעקוף בעיה זו, [נסה תחילה את השלבים](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02) להסרת ההתקנה של יישומי שולחן העבודה של Microsoft Office שהותקנו מראש. 

אם פעולה זו אינה פועלת עבור המחשב המושפע, נסה את [הפתרון המבוסס על רישום](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02).
