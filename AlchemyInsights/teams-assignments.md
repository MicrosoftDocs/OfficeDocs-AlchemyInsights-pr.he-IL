# <a name="teams-assignments"></a>הקצאות צוותים

מטלות הן משימות או יחידות עבודה שהוקצו לתלמיד או חבר צוות בכיתה כחלק מהמחקר שלהם. באפשרותך ליצור מטלות בתוך הכיתה Teams.

- לקבלת מידע נוסף אודות הקצאות, ראה [הקצאות של Microsoft teams](https://support.microsoft.com/en-us/office/microsoft-teams-5aa4431a-8a3c-4aa5-87a6-b6401abea114#ID0EAABAAA=Assignments).
- לקבלת מידע נוסף אודות ניהול תכונות הקצאות במרכז הניהול של Microsoft Teams, ראה [מטלות ב-teams למגזר החינוך](https://docs.microsoft.com/microsoftteams/expand-teams-across-your-org/assignments-in-teams).
