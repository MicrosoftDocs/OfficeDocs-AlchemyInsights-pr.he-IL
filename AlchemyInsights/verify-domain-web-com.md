# <a name="verify-your-domain-with-webcom"></a>אימות התחום שלך באמצעות web.com

כדי לאמת את התחום שלך, יהיה עליך להוסיף רשומת TXT לרשם התחומים כדי לאמת שהתחום נמצא בבעלותך. 

תחילה, בצע את [אשף הוספת התחום](https://portal.office.com/adminportal/home#/Domains) על-ידי בחירת תחומים של **הגדרות** \> **Domains**ולאחר מכן **הוסף תחום**.
  
ראה [יצירת רשומות DNS ב-web.com עבור Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/dns/create-dns-records-at-web-com) לקבלת הוראות שלב-אחר-שלב.
