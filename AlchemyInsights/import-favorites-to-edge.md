# <a name="import-favorites-from-google-chrome-or-mozilla-firefox-to-microsoft-edge"></a>ייבוא מועדפים מ- Google Chrome או ממוזילה Firefox ל- Microsoft Edge

1. פתח את Microsoft Edge ולאחר מכן בחר **הגדרות ועוד ...**  >  **הגדרות**  >  **ייבוא נתוני דפדפן**.

1. בתפריט ייבוא **מתוך,** בחר **Google Chrome או** **Mozilla Firefox** ולאחר מכן בחר **ייבוא**.

1. תחת **פרופיל**, בחר את הפרופיל.

1. תחת **בחר מה לייבא**, בחר את הפריטים שברצונך לייבא ולאחר מכן בחר **ייבוא**.

1. **ב-All done!** תיבת הדו-שיח , בחר **בוצע**.

המועדפים המיובאים צריכים להיות בתיקיה (לדוגמה, מועדפים מיובאים או מועדפים אחרים) סרגל המועדפים. באפשרותך להשאיר אותם במקום שבו הם נמצאים או להעביר אותם לתיקיה אחרת.
