# <a name="distribution-group-not-showing-in-groups-list"></a>קבוצת תפוצה שאינה מוצגת ברשימה ' קבוצות '

הדבר יכול להימשך עד 60 דקות לפני שרשימת התפוצה החדשה תופיע במרכז הניהול.
  
אם ברצונך להוסיף חברים או לבצע פעולות עריכה אחרות כעת, נא השתמש בתכונה ' קבוצות ' [במרכז הניהול של Exchange](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1).
