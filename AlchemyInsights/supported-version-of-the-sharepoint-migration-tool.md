# <a name="supported-version-of-the-sharepoint-migration-tool"></a>גירסה נתמכת של כלי ההעברה של SharePoint

כלי ההעברה של SharePoint תומך כעת רק העברות מ- SharePoint 2013. גירסאות 2010 ו- 2016 אינם נתמכים כרגע.
  
לקבלת מידע נוסף, ראה [הצגת כלי ההעברה של SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
