# <a name="supported-version-of-the-sharepoint-migration-tool"></a>גירסה נתמכת של כלי העברת SharePoint



כיום כלי ההעברה של SharePoint תומך רק בהעברות מ-SharePoint 2013. גירסאות 2010 ו-2016 אינן נתמכות בשלב זה.
  
לקבלת מידע נוסף, ראה [הצגת כלי ההעברה של SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
