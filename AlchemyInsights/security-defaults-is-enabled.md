# <a name="security-defaults-is-enabled"></a>ברירות מחדל של אבטחה מאופשרת

ההגדרה ' ברירות מחדל של אבטחה ' מציגה בקשה לכל המשתמשים לאימות מרובה גורמים.

כדי לנהל ברירות מחדל של אבטחה:

1. היכנס [לפורטל](https://ms.portal.azure.com/)   עם אישורי המנהל הגלובלי שלך.
2. עבור אל מאפייני **active Directory של הספריה**  >  **Properties**.
3. בחלק התחתון של הדף, לחץ על **ברירות מחדל של ניהול אבטחה**.
4. לחץ על **כן** כדי לאפשר ברירות מחדל של אבטחה ועל **לא** כדי להשבית ברירות מחדל של אבטחה.

לקבלת מידע נוסף, ראה [מהם ברירות מחדל של אבטחה?](https://docs.microsoft.com/azure/active-directory/fundamentals/concept-fundamentals-security-defaults).
