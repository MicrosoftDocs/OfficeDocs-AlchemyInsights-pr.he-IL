# <a name="access-denied-due-to-ip-restriction"></a>הגישה נדחתה עקב הגבלת IP

כמנהל IT, באפשרותך לשלוט בגישה למשאבי SharePoint ו-OneDrive ב-Microsoft 365 בהתבסס על מיקומי רשת מוגדרים שאתה נותן בהם אמון. פעולה זו נקראת גם מדיניות מבוססת מיקום. לקבלת מידע נוסף, ראה [שליטה בגישה בהתבסס על מיקום ברשת](https://docs.microsoft.com/sharepoint/control-access-based-on-network-location).
