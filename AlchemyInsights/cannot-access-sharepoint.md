# <a name="cannot-access-sharepoint"></a>לא ניתן לגשת ל-SharePoint

קיימות מספר סיבות לכך שSharePoint או OneDrive עלולים להפוך ללא נגישים. אם אין באפשרותך לגשת ל-SharePoint או OneDrive, השתמש במדריך הבא כדי לפתור בעיה זו.

- [אין אפשרות לגשת אל SharePoint או OneDrive](https://docs.microsoft.com/sharepoint/troubleshoot/sharing-and-permissions/sharepoint-online-inaccessible)
