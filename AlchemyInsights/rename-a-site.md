# <a name="rename-a-sharepoint-site"></a>שינוי שם של אתר SharePoint

בצע את השלבים [במאמר זה](https://docs.microsoft.com/sharepoint/change-site-address) כדי לשנות את שם האתר--לדוגמה https://contoso.sharepoint.com/sites/sample1 , https://contoso.sharepoint.com/sites/sample2מ-ל.

**הערה:** אין באפשרותך לשנות את שם התחום (בדוגמה שלעיל, חלק ה-contoso של כתובת ה-URL) בכתובת URL מקוונת של SharePoint. ראה [שינוי התחומים](https://go.microsoft.com/fwlink/?Linkid=2018696) לקבלת מידע נוסף.
