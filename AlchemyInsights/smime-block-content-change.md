# <a name="outlook-will-now-default-block-external-content-in-smime"></a>Outlook יחסום כעת ברירת מחדל תוכן חיצוני ב-S/MIME

אנו מציגים אמצעי אבטחה שנוספו ב-Outlook עבור Windows על-ידי חסימת תוכן חיצוני בהודעות מוצפנות של S/MIME והודעות חתומות כברירת מחדל ומניעת עקיפות על בסיס כל הודעה.

לקבלת פרטים מלאים, ראה [Outlook עבור Windows חוסם תוכן חיצוני בהודעות S/MIME](https://support.office.com/article/2d3a4af1-fe41-475f-a888-fc7b997d112e).
