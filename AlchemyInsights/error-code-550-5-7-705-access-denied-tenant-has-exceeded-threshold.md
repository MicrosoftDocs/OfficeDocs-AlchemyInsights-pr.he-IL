# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>נדחתה Access 550 5.7.705, דיירים חרג סף

הודעה זו מופיעה כאשר יותר מדי דואר זבל או בצובר שנשלחה על-ידי הארגון שלך Office 365. כדי להסיר אבן זו, עליך להבין מסבירים את הגורם לסוכן תמיכה, כמו גם לתקן את הבעיה הבסיסית.
