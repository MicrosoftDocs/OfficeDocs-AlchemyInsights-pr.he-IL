# <a name="increase-microsoft-365-security"></a>הגדל את מיקרוסופט 365 אבטחה

סקור את [תצורת האבטחה המומלצת](https://docs.microsoft.com/microsoft-365/security/office-365-security/tenant-wide-setup-for-increased-security?view=o365-worldwide) עבור הדייר 365 של Microsoft.

- למד אודות [הציון המאובטח של Microsoft](https://docs.microsoft.com/microsoft-365/security/mtp/microsoft-secure-score?view=o365-worldwide) [ונקוט בפעולה](https://docs.microsoft.com/microsoft-365/security/mtp/microsoft-secure-score?view=o365-worldwide#take-action-to-improve-your-score) כדי **לשפר את התוצאה שלך**.

- למד אודות [תגובה לאירועי אבטחה](https://docs.microsoft.com/microsoft-365/security/office-365-security/office365-security-incident-response-overview?view=o365-worldwide), כולל **חשבונות שנפרצו**.

לקבלת מידע נוסף, ראה [תחילת התחלת עם אבטחה של Microsoft 365](https://docs.microsoft.com/microsoft-365/security/office-365-security/security-roadmap?view=o365-worldwide).
