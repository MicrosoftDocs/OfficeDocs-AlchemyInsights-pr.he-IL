# <a name="change-your-microsoft-account-picture"></a>שינוי תמונת החשבון של Microsoft

1. עבור אל [פרטי חשבון Microsoft שלך](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. בחר באפשרות ' **שנה תמונה**'.
3. בחר **תמונה חדשה**.
4. בחר את התמונה שבה ברצונך להשתמש ובחר באפשרות **פתיחה**.
5. גררו או שנה את גודל תיבת התמונה ובחרו ' **שמור**'.

נא לאפשר עד 24 שעות עבור שינויים בתמונת הפרופיל שלך כדי להיכנס לתוקף.
