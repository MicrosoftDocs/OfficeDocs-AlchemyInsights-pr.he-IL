# <a name="change-your-microsoft-account-picture"></a>שינוי תמונת חשבון Microsoft שלך

1. עבור אל [פרטי חשבון Microsoft שלך](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. בחר **שנה תמונה**.
3. בחר **תמונה חדשה**.
4. בחר את התמונה שבה ברצונך להשתמש ובחר **פתח**.
5. גרור או שנה את הגודל של תיבת התמונה ובחר **שמור**.

אפשר עד 24 שעות לשינוי תמונת הפרופיל שלך כדי להיכנס לתוקף.
