# <a name="delete-a-contact"></a>מחיקת איש קשר

1. בחר בסמל **אנשים** בפינה השמאלית התחתונה של המסך.

2. בחר את איש הקשר שברצונך למחוק ובחר **מחק**.

    אם אין באפשרותך לבחור **למחוק**, איש הקשר ייתכן Skype או חשבון רשת חברתית מחובר. כדי למחוק את איש הקשר, עבור Skype או לחשבון שבו ממוקם איש הקשר.

3. בחר ' **מחק** ' כדי לאשר.
