# <a name="working-with-sharepoint-permission-levels"></a>עבודה עם רמות הרשאה של SharePoint

SharePoint מגיע עם רמות הרשאה המהווים ברירת מחדל שבהן באפשרותך להשתמש, כגון **Contribute** **ותצוגה בלבד**. אך אם אחד מאלה אינו עונה על צרכיך, באפשרותך ליצור רמת הרשאה חדשה. זכור כי קל יותר לעקוב אחר רמות ההרשאה אם אינך משנה את ברירות המחדל.

[כיצד ליצור ולערוך רמות הרשאה](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
