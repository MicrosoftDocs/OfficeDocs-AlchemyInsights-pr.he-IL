<span data-ttu-id="1bfcf-p101">אם תקבל את שגיאה **החשבון שלך הפך ללא זמין** עם `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` כאשר תנסה לפתוח את Outlook באינטרנט (שכונתה בעבר Outlook Web App), מנהל הארגון שלך עשוי ביטל את הגישה שלך ל- Outlook באינטרנט. המנהל שלך יכול לתקן שגיאה זו על-ידי ביצוע השלבים בנושא [לזמינה או ללא זמינה ב- Outlook Web App עבור תיבת דואר](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).</span><span class="sxs-lookup"><span data-stu-id="1bfcf-p101">If you get the error **Your account has been disabled** with  `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` when you try to open Outlook on the web (formerly known as Outlook Web App), your organization admin might have disabled your access to Outlook on the web. Your admin can fix this error by following the steps in the topic [Enable or disable Outlook Web App for a mailbox](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).</span></span>
