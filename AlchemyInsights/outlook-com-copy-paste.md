# <a name="copy-and-paste-in-outlookcom"></a>העתק והדבק ב- Outlook.com

לא קיימים לחצני העתקה או הדבקה ב- Outlook.com, אך באפשרותך להשתמש בקיצורים באמצעות לוח המקשים:

**במחשב:** CTRL + C כדי להעתיק, CTRL + X כדי לגזור ו- CTRL + V כדי להדביק.

**ב- Mac:** CMD + C כדי להעתיק, CMD + X כדי לגזור ו- CMD + V כדי להדביק.
