# <a name="change-language"></a>שנה שפה

1.    היכנס ל- Outlook באינטרנט.
2. בחלק העליון של הדף, בחר **הגדרות** > **להציג את כל ההגדרות של Outlook**.
3. בחר **כללי** > **השפה ואת הזמן**.
4. בחר את השפה שבה ברצונך להשתמש.
5. בחר **שמור**.
 
לקבלת מידע נוסף, ראה [שינוי אזור זמן והגדרות השפה ב- Outlook באינטרנט](https://support.office.com/article/65239869-12e7-4a9d-bca1-76b0ad7ce273).
