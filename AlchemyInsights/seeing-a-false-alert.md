# <a name="seeing-a-false-alert"></a>לראות התראה כוזבת

- שלח קובץ ל-Microsoft לצורך ניתוח. לקבלת מידע נוסף, ראה [מה לעשות עם חיוביים/תשלילים שווא ב-Microsoft Defender אנטי-וירוס](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#submit-a-file-to-microsoft-for-analysis).
- צור מחוון "אפשר" כדי למנוע התראה חיובית שגויה. לקבלת מידע נוסף, ראה [יצירת מחוונים עבור קבצים](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/indicator-file).  
- קבע את תצורת החריגים בהתקן כדי למנוע סריקה של פריט. לקבלת מידע נוסף, ראה [הגדרת הדרה בהתקן בודד של Windows כדי למנוע סריקה של פריט](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#define-an-exclusion-on-an-individual-windows-device-to-prevent-an-item-from-being-scanned).
