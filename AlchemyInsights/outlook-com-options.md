# <a name="how-to-find-options-in-outlookcom-settings"></a>כיצד למצוא אפשרויות בהגדרות Outlook.com

בחלק העליון של הדף, בחר בסמל גלגלי שיניים כדי לפתוח את הגדרות. בחלק התחתון של חלונית הגדרות, בחר [**להציג את כל ההגדרות של Outlook**](https://outlook.live.com/mail/options/general/timeAndLanguage).

פעולה זו תציג את כללי, דואר אלקטרוני, לוח שנה, ו- (אם ישים) Premium הגדרות ואפשרויות.
