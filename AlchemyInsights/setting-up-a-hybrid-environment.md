# <a name="setting-up-a-hybrid-environment"></a>הגדרת סביבה היברידית

כדי לאפשר למשתמשים שלך לעבוד בצורה חלקה ללא קשר אם הנתונים שלהם מאוחסנים בשרתים מקומיים או ב-Microsoft 365, ייתכן שיהיה עליך להגדיר סביבה היברידית. כדי לקבל מבט כולל על האפשרויות שלך, עיין [בשילוב של Microsoft 365 עם סביבות מקומיות](https://docs.microsoft.com/office365/enterprise/office-365-integration).
