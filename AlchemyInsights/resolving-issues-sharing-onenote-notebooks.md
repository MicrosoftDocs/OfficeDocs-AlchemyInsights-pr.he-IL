# <a name="resolving-issues-sharing-onenote-notebooks"></a>פתרון בעיות בשיתוף מחברות של OneNote

לחצן ' שתף ' ממוקם בפינה הימנית העליונה של OneNote.

- הקלד כתובת דואר אלקטרוני אחת כדי לשתף את המחברת.
- לחץ על **שתף**.

סגור ופתח מחדש את המחברת ב-OneNote כדי להתחיל שוב לשתף.
