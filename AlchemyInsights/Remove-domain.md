# <a name="trying-to-remove-your-domain"></a>מנסה להסיר את התחום שלך?

בצע את השלבים במאמר [הסרת תחום מ- Office 365](https://docs.microsoft.com/office365/admin/get-help-with-domains/remove-a-domain).
  
לפני שתסיר את התחום, תצטרך לעדכן את כל המשתמשים או הקבוצות המוקצים לשם תחום זה. עיין ב[שאלות נפוצות בנושא תחומים](https://docs.microsoft.com/office365/admin/setup/domains-faq) למידע על פתרונות נפוצים אחרים.
