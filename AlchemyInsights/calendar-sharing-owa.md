# <a name="troubleshooting-issues-with-calendar-sharing"></a>פתרון בעיות בשיתוף לוח שנה

- כדי לאבחן בעיות הרשאה, השתמש [במנתח הקישוריות המרוחקת של Exchange ללא תשלום](https://testconnectivity.microsoft.com/Default.aspx?testId=freeBusy), אשר מספק הודעות שגיאה ספציפיות מאוד.
- באפשרותך גם להשתמש [במסייע התמיכה והשחזור של microsoft עבור microsoft 365](https://diagnostics.office.com/) כדי לאבחן מגוון של בעיות בלוח השנה. 
- אם המשתמש כולל את Outlook בשולחן העבודה, [כלי הבדיקה של לוח השנה עבור Outlook](https://www.microsoft.com/download/details.aspx?id=28786) יכול לפתור בעיות פוטנציאליות.
