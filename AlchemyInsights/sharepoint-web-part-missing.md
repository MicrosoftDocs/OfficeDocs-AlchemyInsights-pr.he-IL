# <a name="content-editor-or-script-editor-web-parts-are-missing"></a>רכיבי web part של עורך תוכן או עורך סקריפט חסרים

אם חסר לך web part שאתה משתמש בו כדי לראות, למשל **עורך התוכן או ה-web part של עורך הסקריפט**, ייתכן שאתה מנסה להוסיף [web part קלאסי בעמוד מודרני](https://support.office.com/article/classic-and-modern-web-part-experiences-3fdae6c3-8fc1-49ab-8708-8c104b882e64). חשוב לציין שדפים מודרניים יכולים להשתמש רק ברכיבי web part מודרניים ובעמודים קלאסיים ניתן להשתמש רק ברכיבי web part קלאסיים. רכיבי web part מודרניים רבים ממלאים את אותם מטרות כמו רכיבי web part קלאסיים, אך אתה רואה הבדלים במראה ובפונקציונליות.

קיימים רכיבי web part רבים שאינם מופיעים עקב [הגדרת קובץ ה-script המותאמת אישית](https://docs.microsoft.com/sharepoint/allow-or-prevent-custom-script), שאינה זמינה כברירת מחדל באתרים מודרניים. 

לקבלת רשימה של רכיבי web part שאינם זמינים, ראה [תכונות מושפעות כאשר קובץ script מותאם אישית נחסם](https://docs.microsoft.com/sharepoint/allow-or-prevent-custom-script#features-affected-when-custom-script-is-blocked).

לקבלת מידע נוסף אודות ההשלכות הביטחוניות של קובץ script מותאם אישית, ראה [שיקולי אבטחה להתרת תסריט מותאם אישית](https://docs.microsoft.com/sharepoint/security-considerations-of-allowing-custom-script).
