# <a name="create-a-folder"></a>צור תיקיה

1. בחלונית הימנית, בחר **צור תיקיה חדשה**![ליצור לחצן תיקיה חדשה](media/d8e28612-fbdb-4d28-a4d0-14f7834cfd97.png) לצד **תיקיות**. 
    
2. בתיבה תיקיה חדשה, הקלד שם עבור התיקיה ולאחר מכן הקש Enter.
    
לקבלת מידע נוסף, ראה [יצירת תיקיה ב- Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873114).
